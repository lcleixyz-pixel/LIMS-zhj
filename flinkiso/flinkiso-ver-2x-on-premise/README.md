
# FlinkISO Open Source QMS

FlinkISO uses CakePHP 2.10.x version as a MVC architecture framework. In order to install and run FlinkISO, you must follow all the directions available on CakePHP website:

[All the prerequisite & installation process](https://book.cakephp.org/2/en/installation.html). 

Once your server is setup, you can install FlinkISO. To Install FlinkISO, follow the installation manual available: 

[https://www.flinkiso.com/manual/installation.html](https://www.flinkiso.com/manual/installation.html) - How to install & run application, import schema.sql, database connection, email configurations, directory permissions etc.

Note: 
1. When you download the application and if you are using a older versions of PHP (PHP<8) then you must rename the lib<8 to lib for application to function on older versions of PHP.
1. You must have Static IP/ Web-facing URL for ONLYOFFICE from FlinkISO server. ONLYOFFICE will not run on localhost. To avoid static IP, you must to install your own copy of ONLYOFFICE Editors.You must install ONLYOFFICE on a seperate server. Application will not work if you install ONLYOFFICE Editors & FlinkISO on a same server. You can use VM to install ONLYOFFICE Editors.

## ONLYOFFICE
You can download and install ONLYOFFICE Editors from [https://www.onlyoffice.com/download-docs.aspx#docs-community](https://www.onlyoffice.com/download-docs.aspx#docs-community)

## FlinKISO community Forum 
For free community support, visit [https://forum.flinkiso.com](https://forum.flinkiso.com). In order to add post, you must be a registered On-Premise User. A token will be sent to you at your registered email address with us. Use that token for login. You can generate a new token whenever you wish to.
