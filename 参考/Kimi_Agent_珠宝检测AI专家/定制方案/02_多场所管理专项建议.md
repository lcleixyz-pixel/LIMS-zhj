## 第一部分：多场所管理体系架构设计

### A. 组织架构模式选择

#### 模式A：集中式（Centralized）— 推荐方案

| 项目 | 说明 |
|------|------|
| **质量负责人** | 统一设置1人，驻主场所，全面负责两场所质量管理体系 |
| **技术负责人** | 统一设置1人，驻主场所，全面负责两场所技术运作 |
| **分场所管理员** | 设分场所质量/技术管理员各1人（可兼任），执行日常管理 |
| **决策模式** | 重大事项统一决策，分场所执行 |
| **文件控制** | 统一质量手册+统一程序文件，分场所特殊要求以《场所附件》形式补充 |

#### 模式B：分散式（Decentralized）

| 项目 | 说明 |
|------|------|
| **质量负责人** | 各场所独立设置（共2人） |
| **技术负责人** | 各场所独立设置（共2人） |
| **决策模式** | 各场所相对独立，重大事项协商决策 |
| **文件控制** | 各场所独立体系文件，主场所仅做协调 |

#### 推荐方案：模式A（集中式）及理由

| 考量因素 | 模式A（集中式） | 模式B（分散式） | 推荐结论 |
|----------|-----------------|-----------------|----------|
| 人员规模适配性 | 20人规模下更经济高效 | 需至少22-24人才能合理配置 | **模式A胜出** |
| 体系一致性 | 统一标准，两场所结果可比性强 | 易出现两场所"两张皮" | **模式A胜出** |
| CNAS评审要求 | 统一质量/技术负责人，满足CL01 G001 4.1.5 | 需证明两场所协调有效 | **模式A胜出** |
| CMA监管合规 | 满足《评审准则》第十条多场所统一要求 | 各场所独立取证，成本高 | **模式A胜出** |
| 运营效率 | 统一采购、统一培训、资源共享 | 重复投入，资源分散 | **模式A胜出** |
| 风险管控 | 统一监控，风险集中管控 | 分散管控，风险点增多 | **模式A胜出** |

> **法规依据**：CNAS-CL01:2018第4.1.5条"实验室应规定其组织和管理结构"；CNAS-RL01:2025第8.1.2条"各场所内部的组织机构和人员职责应明确"

### B. 关键岗位设置

#### 总体岗位架构图

```
                    ┌─────────────────┐
                    │   最高管理者    │  1人（统一）
                    │   （主任）      │     驻主场所
                    └────────┬────────┘
                             │
          ┌──────────────────┼──────────────────┐
          │                  │                  │
    ┌─────▼─────┐    ┌─────▼─────┐    ┌─────▼─────┐
    │  质量负责人 │    │  技术负责人 │    │  管理层代表 │
    │   1人      │    │   1人      │    │  （可兼任） │
    │  驻主场所   │    │  驻主场所   │    │            │
    └─────┬─────┘    └─────┬─────┘    └────────────┘
          │                  │
          │    ┌─────────────┘
          │    │
    ┌─────▼────▼─────┐
    │   分场所管理    │
    │  质量/技术管理员 │  1人（驻分场所，可兼）
    └─────────────────┘
```

#### 各岗位详细说明

| 岗位 | 人数 | 场所安排 | 统一/分设 | 依据条款 | 兼职建议 |
|------|------|----------|-----------|----------|----------|
| **最高管理者** | 1人 | 驻主场所 | **统一** | CNAS-CL01:2018 4.1.5 | 不建议兼职 |
| **质量负责人** | 1人 | 驻主场所 | **统一** | CNAS-CL01:2018 4.1.5g) | 可兼管理层代表 |
| **技术负责人** | 1人 | 驻主场所 | **统一** | CNAS-CL01:2018 4.1.5h) | 可兼授权签字人 |
| **授权签字人** | 主2+分1=3人 | 主场所2人、分场所1人 | **各场所独立配置** | CNAS-RL01:2025 7.3 | 可由技术负责人兼任主场所签字人 |
| **分场所质量/技术管理员** | 1人 | 驻分场所 | 分场所设执行岗 | CNAS-RL01:2025 8.1.2 | 可由分场所检测骨干兼任 |

#### ⚠️ 关键提醒

1. **质量负责人不可兼职其他实验室岗位**：CNAS-RL01:2025第7.7条明确规定"从事检测/校准/鉴定活动的人员不得同时在其他实验室兼职"，但质量负责人在本机构内可适当兼职（需确保履职时间充足）。

2. **分场所是否需独立质量/技术负责人？**  
   对于20人规模的两场所实验室：  
   - CNAS未强制要求分场所独立设置质量/技术负责人  
   - 但CNAS-RL01:2025第8.1.2条要求"需要时，应配备相应的质量和技术管理人员"  
   - **建议**：分场所设1名质量/技术管理员（可由授权签字人或资深检测员兼任），负责日常体系运行联络，向主场所质量/技术负责人汇报

### C. 体系文件架构

#### 文件层次结构

```
第一层：质量手册（统一版本）
├── 通用部分：覆盖两场所的共性要求
├── 场所附件A：主场所特殊说明
├── 场所附件B：分场所特殊说明
└── 适用性声明：明确各条款对两场所的适用性

第二层：程序文件
├── 统一程序文件（两场所共用，约20-25个）
│   ├── JD-CX-01 文件控制程序
│   ├── JD-CX-02 记录控制程序
│   ├── JD-CX-03 管理评审程序
│   ├── ...
│   └── JD-CX-25 数据控制和信息管理程序
└── 分场所特殊程序（必要时制定）
    ├── JD-CX-FS-01 分场所样品管理补充规定
    └── JD-CX-FS-02 分场所设备期间核查补充规定

第三层：作业指导书
├── 统一作业指导书（两场所共用）
│   ├── JD-ZY-01 贵金属XRF检测作业指导书
│   ├── JD-ZY-02 珠宝玉石鉴定作业指导书
│   └── ...
└── 分场所特殊作业指导书
    └── JD-ZY-FS-01 分场所XRF设备操作规程

第四层：记录表格
├── 统一记录表格（两场所共用）
└── 分场所专用记录表格
```

#### 质量手册场所附件内容模板

📋 **主场所附件A应包含**：
- 主场所地址、联系方式
- 主场所组织架构及人员配置
- 主场所设备清单（关键设备）
- 主场所环境条件说明
- 主场所检测能力范围
- 主场所授权签字人名单

📋 **分场所附件B应包含**：
- 分场所地址、联系方式
- 分场所组织架构及人员配置
- 分场所与主场所的管理关系说明
- 分场所设备清单（关键设备）
- 分场所环境条件说明
- 分场所检测能力范围
- 分场所授权签字人名单
- 分场所样品流转路径说明

#### 程序文件的统一与差异化

| 程序文件类别 | 统一/差异 | 说明 | 示例 |
|-------------|-----------|------|------|
| **管理类程序** | 统一 | 两场所共同执行 | 文件控制、记录控制、内部审核、管理评审、纠正措施 |
| **技术类程序-通用部分** | 统一 | 方法原理一致 | 方法验证、不确定度评定、能力验证、分包管理 |
| **技术类程序-设备部分** | 统一+差异附件 | 设备型号可能不同 | 设备管理（主附件A、分附件B） |
| **样品管理程序** | 统一+差异附件 | 流转路径不同 | 样品管理（注明主/分场所流转差异） |
| **报告管理程序** | 统一+差异附件 | 授权签字人不同 | 报告控制（分场所报告需标注分场所信息） |

## 第二部分：CMA多场所管理要点

### 2.1 分场所CMA资质的法律要求

| 要求项 | 具体内容 | 法规依据 |
|--------|----------|----------|
| **法律地位** | 分场所应为同一法人实体，具有明确法律地位 | 《评审准则》第八条 |
| **场所要求** | 具有符合标准要求的检验检测场所，包括多个地点的场所 | 《评审准则》第十条(一) |
| **独立设备** | 各场所应配备具有独立支配使用权的设备和设施 | 《评审准则》第十一条(一) |
| **技术能力** | 各场所的技术能力应分别进行确认 | 《评审准则》第十五条 |
| **管理体系** | 建立保证检验检测活动独立、公正、科学、诚信的管理体系 | 《评审准则》第十二条 |

> **法规依据**：《检验检测机构资质认定评审准则》（市场监管总局公告2023年第21号）第十条明确规定"检验检测机构具有符合标准或者技术规范要求的检验检测场所，包括固定的、临时的、可移动的或者**多个地点的场所**"

### 2.2 多场所管理体系覆盖要求

| 覆盖要素 | 主场所 | 分场所 | 统一要求 |
|----------|--------|--------|----------|
| **质量手册** | 适用 | 适用 | 统一版本+场所附件 |
| **程序文件** | 适用 | 适用 | 统一版本（特殊要求以附件补充） |
| **作业指导书** | 适用 | 适用 | 统一版本 |
| **内审** | 每年至少1次 | 每年至少1次 | 统一策划，分别实施 |
| **管理评审** | 每年至少1次 | 纳入主场所管理评审 | 统一实施，覆盖所有场所 |
| **人员培训** | 适用 | 适用 | 统一计划，分场所实施 |
| **设备管理** | 独立管理 | 独立管理 | 统一制度，各场所分别执行 |
| **质量控制** | 适用 | 适用 | 统一策划，各场所分别实施 |

### 2.3 分场所的CMA监督评审安排

| 评审类型 | 覆盖范围 | 具体要求 |
|----------|----------|----------|
| **首次评审** | 所有场所全覆盖 | 现场试验覆盖所有场所的关键项目/参数 |
| **复查换证评审** | 所有场所全覆盖 | 所有检验检测场所均需接受评审 |
| **扩项评审** | 涉及扩项的场所 | 新增能力的场所必须现场评审 |
| **变更评审** | 涉及变更的场所 | 地址变更、能力变更的相关场所 |
| **远程评审** | 可选择适用 | 《评审准则》第十七条：多场所从事完全相同检测活动可适用远程评审 |

> ⚠️ **重要提醒**：根据《评审准则》第十五条，"安排现场试验时应当覆盖所有申请类别的主要或关键项目/参数、仪器设备、检测方法、试验人员、试验材料等，并**覆盖所有检验检测场所**。"

## 第三部分：CNAS多场所认可要点

> **核心法规**：CNAS-RL01:2025《实验室认可规则》第8章"对多场所实验室认可的特殊要求"

### 3.1 管理体系覆盖要求

| 条款 | 要求内容 | 实施要点 |
|------|----------|----------|
| **8.1.1** | 管理体系应覆盖开展检测活动或与检测活动相关的所有场所 | 质量手册、程序文件须明确覆盖两场所；内审和管理评审须覆盖所有场所 |
| **8.1.2** | 各场所内部的组织机构和人员职责应明确，需要时配备质量和技术管理人员 | 分场所设质量/技术管理员1人（可兼），明确岗位职责 |
| **8.1.3** | 各场所参加能力验证的频次和覆盖领域应满足CNAS-RL02要求 | 主、分场所分别参加能力验证，均须满足要求 |

### 3.2 各场所管理体系运行6个月以上

⚠️ **这是CNAS多场所认可的核心硬性要求**

| 要求项 | 具体内容 | 注意事项 |
|--------|----------|----------|
| **运行时间** | 各场所管理体系正式运行**6个月以上** | 分场所从发布体系文件、人员到位、设备配置齐全开始计算 |
| **内审要求** | 各场所须进行覆盖该场所**所有活动**的内部审核 | 分场所须独立完成至少1次全要素内审 |
| **管理评审** | 各场所须进行过覆盖该场所**所有活动**的管理评审 | 可纳入主场所统一管理评审，但须覆盖分场所所有活动 |

**📋 分场所体系运行时间线示例**

```
第1个月：分场所人员到位、设备到位、体系文件发布
第2个月：试运行，开展人员培训、设备校准
第3个月：试运行，开展检测工作，积累运行记录
第4个月：试运行，进行日常质量监督
第5个月：分场所内部审核（第1次）
第6个月：管理评审（覆盖分场所）
第6个月末：具备申请CNAS认可条件
```

> **法规依据**：CNAS-RL01:2025第8.2.3条"申请认可时，各个开展检测/校准/鉴定活动的场所其管理体系均应正式运行6个月以上，且进行过覆盖该场所所有活动的内审和管理评审。"

### 3.3 各场所独立内审和管理评审要求

#### 内部审核要求

| 项目 | 主场所 | 分场所 |
|------|--------|--------|
| **审核频次** | 每年至少1次 | 每年至少1次 |
| **审核范围** | 覆盖本场所所有要素和活动 | 覆盖本场所所有要素和活动 |
| **审核方式** | 可独立进行 | 可独立进行或由主场所审核组统一进行 |
| **审核员** | 经培训合格的内审员 | 可安排主场所内审员交叉审核 |
| **审核报告** | 独立出具 | 独立出具或作为统一报告的分场所章节 |

#### 管理评审要求

| 方式 | 说明 | 推荐度 |
|------|------|--------|
| **统一评审** | 主场所组织管理评审会议，分场所负责人参加或提交书面报告，统一管理评审报告 | ★★★★★ |
| **分别评审** | 各场所分别进行管理评审，分别出具报告 | ★★☆☆☆ |
| **混合评审** | 统一策划、分别输入、统一评审、统一输出 | ★★★★☆ |

**推荐方式：统一评审模式**

| 管理评审输入项 | 主场所提供 | 分场所提供 |
|---------------|------------|------------|
| 政策和程序的适用性 | 主场所分析 | 分场所分析 |
| 监督结果 | 主场所监督报告 | 分场所监督报告 |
| 内部审核结果 | 主场所内审报告 | 分场所内审报告 |
| 纠正措施 | 主场所纠正措施 | 分场所纠正措施 |

| **比对样品** | 贵金属监控样品/留样 |
| **比对频率** | 每季度至少1次 |
| **参与人员** | 主场所检测员1名 + 分场所检测员1名 |
| **比对方法** | 同一样品分别在两场所检测，各测3次取均值 |
| **判定标准** | 两场所结果偏差≤±0.5%为满意 |
| **记录保存** | 比对报告保存至认可周期结束后6年 |
| **不满意处理** | 偏差>±0.5%时，分析原因，采取纠正措施 |

## 第六部分：风险与应对

### 6.1 多场所管理常见风险点（10项）

| 序号 | 风险类别 | 风险描述 | 风险等级 | 法规依据 |
|------|----------|----------|----------|----------|
| **R1** | 体系覆盖不全 | 分场所管理体系运行记录不完整，未覆盖所有活动 | **高** | CNAS-RL01:2025 8.2.3 |
| **R2** | 技术能力差异 | 两场所检测结果不一致，但无法追溯原因 | **高** | CNAS-RL01:2025 8.3.2 |
| **R3** | 人员兼职冲突 | 分场所关键岗位人员同时承担过多职责，履职不到位 | **中** | CNAS-RL01:2025 7.7 |
| **R4** | 设备管理失控 | 分场所设备校准过期、期间核查记录缺失 | **高** | CNAS-CL01:2018 6.4 |
| **R5** | 能力验证遗漏 | 分场所未独立参加能力验证，或结果不满意未处理 | **高** | CNAS-RL02:2023 |
| **R6** | 样品管理混乱 | 跨场所流转的样品状态失控，标识不清 | **中** | CNAS-CL01:2018 7.4 |
| **R7** | 标准物质过期 | 分场所标准物质证书过期仍继续使用 | **高** | CNAS-CL01-A015 6.5.2 |
| **R8** | 内审管理评审形式化 | 分场所内审/管理评审走过场，未真正发现问题 | **中** | CNAS-RL01:2025 8.2.3 |
| **R9** | 授权签字人越权 | 授权签字人签发非授权领域的报告 | **高** | CNAS-RL01:2025 7.3 |
| **R10** | 档案管理混乱 | 分场所检测档案不完整，无法追溯 | **中** | CNAS-CL01:2018 7.5 |

### 6.2 应对措施

| 风险编号 | 应对措施 | 责任人 | 完成时限 | 检查方式 |
|----------|----------|--------|----------|----------|
| **R1** | 制定《分场所体系运行检查表》，每月由主场所质量部对分场所进行检查 | 质量负责人 | 每月 | 检查记录 |
| **R2** | 建立两场所定期比对机制（每季度1次），统一质量控制样品 | 技术负责人 | 每季度 | 比对报告 |
| **R3** | 制定分场所岗位职责说明书，明确各岗位工作时间分配 | 最高管理者 | 体系建立时 | 岗位说明书 |
| **R4** | 建立统一设备管理台账，设置校准到期前1个月预警 | 设备管理员 | 持续 | 设备台账 |
| **R5** | 制定年度能力验证计划，主分场所分别列入，跟踪完成情况 | 质量负责人 | 每年1月 | 能力验证计划表 |
| **R6** | 制定《跨场所样品流转管理规定》，明确流转记录要求 | 技术负责人 | 体系建立时 | 流转记录 |
| **R7** | 建立标准物质管理台账，设置有效期到期前3个月预警 | 设备管理员 | 持续 | 标准物质台账 |
| **R8** | 主场所审核组成员参与分场所内审，提高内审质量 | 质量负责人 | 每次内审 | 内审报告 |
| **R9** | 明确各授权签字人签字领域，在报告中设置签字领域限定 | 质量负责人 | 体系建立时 | 授权签字人一览表 |
| **R10** | 制定《档案管理规定》，分场所档案每季度移交主场所存档 | 质量负责人 | 每季度 | 档案清单 |

### 6.3 ⚠️ 特别容易出问题的高风险环节

#### ⚠️ 高风险环节一：分场所管理体系运行时间不足

| 项目 | 说明 |
|------|------|
| **问题描述** | 分场所体系运行未满6个月就提交CNAS申请，导致不予受理 |
| **法规要求** | CNAS-RL01:2025第8.2.3条"各场所管理体系均应正式运行6个月以上" |
| **解决方案** | 分场所体系文件发布后，严格按6个月运行期执行，提前准备运行记录 |
| **关键记录** | 文件发放记录、培训记录、设备使用记录、检测原始记录、监督记录、内审报告、管理评审报告 |

#### ⚠️ 高风险环节二：两场所检测结果不一致

| 项目 | 说明 |
|------|------|
| **问题描述** | 同一样品在主、分场所检测结果偏差超过允许范围 |
| **可能原因** | 设备校准状态不同、操作手法差异、环境条件影响、标准物质差异 |
| **解决方案** | 建立定期比对机制；统一操作规程；统一标准物质；统一环境控制 |
| **应急措施** | 发现偏差立即停止检测→排查原因→设备重新校准→人员重新培训→比对合格后方可恢复 |

#### ⚠️ 高风险环节三：能力验证覆盖不足

| 项目 | 说明 |
|------|------|
| **问题描述** | 分场所未独立参加能力验证，或参加领域未覆盖全部认可范围 |
| **法规要求** | CNAS-RL02:2023要求每个场所分别满足能力验证要求 |
| **解决方案** | 每年初制定能力验证计划，明确主、分场所各自参加的PT项目，跟踪完成情况 |
| **特别注意** | 分场所不能以主场所的能力验证结果代替，必须独立参加 |

#### ⚠️ 高风险环节四：授权签字人配置不当

| 项目 | 说明 |
|------|------|
| **问题描述** | 分场所无授权签字人，或授权签字人同时在多场所签字 |
| **法规要求** | CNAS-RL01:2025第7.3条要求授权签字人"在对检测/校准/鉴定结果的正确性负责的岗位上任职" |
| **解决方案** | 分场所必须配置至少1名授权签字人；授权签字人须明确签字领域和场所 |
| **特别注意** | 授权签字人变更须在20个工作日内通知CNAS（RL01:2025第9.1.1条） |

#### ⚠️ 高风险环节五：分场所内审和管理评审形式化

| 项目 | 说明 |
|------|------|
| **问题描述** | 分场所内审只走过场，管理评审输入不充分，未真正识别问题 |
| **法规要求** | CNAS-RL01:2025第8.2.3条要求"进行过覆盖该场所所有活动的内审和管理评审" |
| **解决方案** | 主场所审核员参与分场所内审；管理评审输入须包含分场所数据；分场所负责人参加管理评审会议 |
| **关键检查点** | 内审检查表是否覆盖分场所所有要素；管理评审输入是否包含分场所运行数据 |

### 6.4 多场所管理关键时间节点总表

| 阶段 | 时间节点 | 任务 | 责任人 |
