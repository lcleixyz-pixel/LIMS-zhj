## 1. CNAS-CL01:2018结构概述

### 1.1 准则定位与适用范围

## 2. 第4章 通用要求

### 2.1 公正性（4.1）

实验室应公正地实施实验室活动，并从组织结构和管理上保证公正性。具体要求包括：

- **4.1.1** 实验室或其所属机构应是法定机构登记注册的法人机构
- **4.1.2** 实验室管理层对公正性做出承诺
- **4.1.3** 实验室应对实验室活动的公正性负责，不允许商业、财务或其他压力损害公正性
- **4.1.4** 实验室应持续识别影响公正性的风险（G001:2024强调"持续不断"的识别）
- **4.1.5** 当识别出公正性风险时，实验室应能证明其如何消除或最大限度降低此类风险

```

Source: CNAS-CL01-G001应用要求

### 2.2 保密性（4.2）

- **4.2.1** 实验室应通过具有法律效力的承诺，对在实验室活动中获得或产生的所有信息承担管理责任
- **4.2.2** 实验室依据法律要求或合同授权透露保密信息时，应将所提供的信息通知到相关客户
- **4.2.3** 从客户以外渠道获取的信息应在客户和实验室间保密
- **4.2.4** 所有人员（包括委员会成员、合同方、外部机构等）应对在实施实验室活动过程中获得或产生的所有信息保密

## 3. 第5章 结构要求

第5章结构要求主要明确实验室的法律责任和管理框架：

- **5.1** 实验室或其母体机构应是能承担法律责任的实体
- **5.2** 实验室管理层对实验室活动全权负责
- **5.3** 实验室应规定符合本准则要求的实验室活动范围
- **5.4** 实验室应以满足本准则、实验室客户、法定管理机构和提供承认的组织要求的方式开展实验室活动
- **5.5** 实验室管理层应提供建立和实施管理体系以及持续改进其有效性承诺的证据
- **5.6** 实验室应明确管理层人员的职责和权限

## 4. 第6章 资源要求

### 4.1 总则（6.1）

实验室应获得管理和实施实验室活动所需的所有资源。

### 4.2 人员（6.2）

人员要求是资源要求中的核心要素：

| 条款 | 关键要求 |
|------|----------|
| 6.2.1 | 所有可能影响实验室活动的人员应具备能力 |
| 6.2.2 | 实验室应将每个影响实验室活动的人员的能力要求形成文件 |
| 6.2.3 | 实验室应确保人员具备相应的能力，在授权前进行确认 |
| 6.2.4 | 实验室应监控人员的能力，通过培训、教育、监督等方式 |
| 6.2.5 | 实验室应授权特定人员：抽样、检测/校准、签发报告、设备操作等 |

**G001:2024对人员的要求细化：**
- 从事检测或校准活动的人员不得在其他同类型实验室从事同类活动
- 关键技术人员需有3年以上本专业领域经历
- 授权签字人需熟悉CNAS认可要求，具有中级以上技术职称或同等能力
- 同等能力条件：大专毕业后7年以上；本科毕业后5年以上；硕士毕业后2年以上

### 4.3 设施和环境条件（6.3）

- **6.3.1** 实验室的设施应为自有设施，拥有全部使用权和支配权
- 自有设施指购买或长期租赁（至少覆盖一个认可周期）的设施
- 实验室应确保环境条件不会使结果无效或对结果产生不利影响
- 应对环境条件进行监控、控制和记录

### 4.4 设备（6.4）

设备管理是实验室能力的重要保障：

| 条款 | 关键要求 |
|------|----------|
| 6.4.1 | 实验室应获得正确开展实验室活动所需的设备 |
| 6.4.2 | 用于测量的设备应能达到要求的测量准确度和符合相应的计量要求 |
| 6.4.3 | 设备应由经过授权的人员操作 |
| 6.4.4 | 实验室应对每台设备建立唯一性标识和记录 |
| 6.4.5 | 实验室应保存设备的校准/检定记录 |
| 6.4.6 | 应建立机制，提示对到期设备进行校准、核查和维护 |
| 6.4.7 | 需要期间核查的设备应确定核查方法与周期 |
| 6.4.8 | 设备的校准状态应通过标签、编码等方式标识 |
| 6.4.9 | 设备应得到保护，防止调整导致结果失效 |
| 6.4.10 | 设备故障应停止使用并标识 |
| 6.4.11 | 软件视为设备，应满足设备管理要求 |

### 4.5 计量溯源性（6.5）

- 实验室应确保测量结果能溯源至SI单位
- 应通过具备能力的校准实验室进行溯源
- 溯源证书应包含测量结果和测量不确定度
- 当无法溯源至SI单位时，应溯源至参考物质(RM)或公认的方法

### 4.6 外部提供的产品和服务（6.6）

- 实验室应确保外部提供的产品和服务符合要求
- 应对外部供应商进行评价、监控表现和再次评价
- 能力验证/测量审核提供者也应列入合格供应商管理

## 5. 第7章 过程要求

### 5.1 要求、标书和合同评审（7.1）

- 实验室应建立评审客户要求、标书和合同的程序
- 评审应确保实验室有能力满足客户要求
- 必要时给客户提供充分说明，以便客户申请适合自身需求的检测项目

### 5.2 方法的选择、验证和确认（7.2）

这是技术核心条款：

| 条款 | 关键要求 |
|------|----------|
| 7.2.1 | 优先使用国际标准、区域标准或国家标准方法 |
| 7.2.2 | 引入标准方法前应进行验证，验证内容包括精密度、线性范围、检出限等 |
| 7.2.3 | 非标方法应进行确认，确认记录应保存 |
| 7.2.4 | 方法偏离需文件化、技术判断批准、客户同意 |

**G001:2024对方法验证要求升级：**
- 验证范围扩大：新增对非标准方法、实验室自制方法的验证要求
- 准确度验证：要求低、中、高3个浓度水平（2018版仅1个）
- 精密度验证：增加实验室间比对要求
- 动态验证机制：建议周期不超过2年进行再验证

```
Claim: G001:2024对方法验证要求升级，新增准确度3个浓度水平验证、实验室间比对、动态验证机制（周期不超过2年）

### 5.3 抽样（7.3）

- 仅进行抽样而不从事后续检测/校准的，CNAS不认可该抽样项目
- 从客户提供的样品中取出部分样品进行检测时，应有书面的取样程序或记录
- 确保样品的均匀性和代表性

### 5.4 检测或校准物品的处置（7.4）

- 应建立样品的标识系统，确保样品不混淆
- 样品标识不应粘贴在容易与盛装样品容器分离的部件上
- 处理程序应保障客户信息安全，明确客户的所有权

### 5.5 技术记录（7.5）

技术记录应包含以下信息：
- 样品描述和唯一性标识
- 检测/校准和抽样方法
- 环境条件
- 设备和标准物质信息
- 检测过程中的原始观察记录、计算
- 实施实验室活动的人员

```

Confidence: high
```

### 5.6 测量不确定度的评定（7.6）

- 实验室应对所有检测/校准项目评估测量不确定度
- 在以下情况需在报告中报告不确定度：客户要求、影响符合性判定、标准要求
- 评定方法参考GUM法（Guide to the Expression of Uncertainty in Measurement）

### 5.7 确保结果有效性（7.7）

质量监控要求：
- 质量监控计划应覆盖认可范围内的所有检测/校准项目
- 内部质量监控：质控图、标准物质、留样再测、人员比对、设备比对等
- 外部质量监控：能力验证(PT)、实验室间比对
- CNAS-AL07《能力验证领域和频次表》是最低要求

### 5.8 报告结果（7.8）

报告要求：
- 报告应在同一份上出具特定样品不同检测项目的结果
- 客户送样应在报告中明确标注"客户送样"
- 报告应按GB/T 8170进行数值修约
- 做出符合性声明时建议参考CNAS-GL27
- 意见和解释需明确是否在认可范围内

### 5.9 投诉（7.9）

- 实验室应建立处理投诉的程序
- 投诉处理过程应文件化
- 收到投诉后应确认是否与其负责的实验室活动相关

### 5.10 不符合工作（7.10）

不符合工作的处理流程：
1. 识别不符合工作
2. 确定不符合工作的可接受性
3. 必要时暂停工作和扣发报告
4. 采取纠正措施
5. 评价纠正措施的有效性
6. 恢复工作

### 5.11 数据控制和信息管理（7.11）

- 使用LIMS时，需确保满足审核路径、数据安全和完整性要求
- 应对LIMS与认可要求的符合性和适宜性进行完整确认
- 确保LIMS的改进和维护可获取先前产生的记录

## 6. 第8章 管理体系要求

### 6.1 方式（8.1）

实验室可选择以下方式建立管理体系：
- **方式A**：按本准则第8.2-8.9条款建立完整的管理体系
- **方式B**：按ISO 9001建立管理体系，并证明满足本准则的技术要求

### 6.2 管理体系文件（8.2）

- 质量手册、程序文件、作业指导书、记录表格
- 若实验室是某个组织的一部分，需提炼汇总相关实验室规定，形成管理体系文件
- 质量手册应由对实验室和相关部门承担管理职责的该组织的负责人批准

### 6.3 管理体系文件的控制（8.3）

- 文件发布前经授权人员审批
- 定期评审和必要时的更新
- 确保使用场所获得适用文件的有关版本
- 文件应唯一性标识

### 6.4 记录控制（8.4）

**G001:2024对记录保存期限的明确规定：**

| 记录类型 | 保存期限 |
|----------|----------|
| 技术记录（检测/校准/抽样原始记录） | 至少6年 |
| 人员或设备记录 | 人员工作期间/设备使用时限全程保留，调离或停用后至少再保存6年 |

## 11. 研究发现总结

### 11.1 核心发现

1. **CNAS-CL01:2018是实验室认可的基石文件**，等同采用ISO/IEC 17025:2017，覆盖通用要求、结构要求、资源要求、过程要求和管理体系要求五大模块。

2. **G001:2024是2024-2025年最重要的政策变化**，2025年7月1日起强制执行，新增7项要求、变更38项条款，重点强化风险管理、数据完整性、设备管理和网络安全。

3. **珠宝检测领域有专门的应用说明A015:2018**，对人员资质（宝石学背景）、环境条件（23℃±1℃）、设备配置（XRF/折射仪等）、方法验证（GB/T 18043系列）有特殊要求。

4. **认可流程通常需要6-12个月**，包括体系建立（3-6月）、试运行（至少6月）、申请评审（3-6月）和整改发证（1-3月）。

5. **RL01:2025认可规则强化了动态管理**，新增专项监督机制、细化变更通报要求、明确整改时限（严重不符合1个月、一般不符合2个月）。
