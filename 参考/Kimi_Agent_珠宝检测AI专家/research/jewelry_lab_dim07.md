# 维度7：检验检测机构体系文件编写与管理

## 研究概述

本研究系统梳理了检验检测机构质量管理体系文件的架构设计、编写方法和管理要求，涵盖质量手册、程序文件、作业指导书、记录表格四个层次，以及文件控制的全流程管理。特别关注珠宝检测实验室在体系文件建设中的特殊要求，为珠宝检测实验室建立和完善管理体系提供全面参考。

---

## 一、体系文件架构

### 1.1 四级文件结构

检验检测机构的管理体系文件通常采用金字塔式四级结构，各层次文件相互关联、层层支撑：

| 层级 | 文件类型 | 作用定位 | 使用对象 |
|------|----------|----------|----------|
| 第一层次 | 质量手册 | 纲领性文件，阐述质量方针、目标及管理体系整体架构 | 整个机构及领导层 |
| 第二层次 | 程序文件 | 支持性文件，描述跨部门活动的流程与职责 | 机构内各部门 |
| 第三层次 | 作业指导书(SOP) | 操作性文件，提供具体检测项目或设备操作的技术细则 | 具体操作人员 |
| 第四层次 | 记录表格 | 证实性文件，保存体系运行的证据 | 相关岗位人员 |

### 1.2 各层级文件关系

上下层次文件要相互衔接，质量手册要求原则，程序文件规定具体，作业指导书技术性、实际操作性强。质量手册：质量体系是顶层文件，质量手册中规定了数个质量管理活动，每个活动制定一个控制文件，就是"程序文件"。程序文件更多涉及到工作步骤，希望通过对过程的约定，实现质量活动的可控。作业指导书(SOP)更具体，可以细化到完成某个工作任务的"动作"。质量记录以表单形式做好各项记录，主要是为了体系运行的过程追溯留下记录。

### 1.3 文件编制原则

文件编制应遵循以下原则：
1. **符合性**：应符合并覆盖《评审准则》的所有要求
2. **适用性**：应符合本检测机构的实际情况，具体的控制要求应以满足机构需要为准
3. **协调性**：文件和文件之间应相互协调，避免文件之间的冲突
4. **完整性**：文件的相关信息齐全，通常应包括文件唯一性标识（编号）、文件名称、编制、审核、批准、实施日期、受控状态、分发号、版本号、修改次数、页码、总页数

## 二、质量手册编写

### 2.1 质量手册的定位与作用

质量手册是实验室质量管理体系的顶层设计文件，其编写质量直接关系到整个体系的有效性。质量手册的作用主要包括四个方面：
1. 管理体系要素文件化，识别和规定实验室的主要过程
2. 明确实验室的方针、目标，全体人员达成共识
3. 明确实验室人员的职责权限关系及各项政策制度
4. 作为质量保证提供给客户、认可机构及相关方

### 2.2 质量手册的结构

一份完整的质量手册应当包含以下组成部分：

| 章节 | 内容要求 |
|------|----------|
| 封面 | 机构名称、手册名称、文件编号、版本号、发布日期、受控编号 |
| 修订页 | 序号、修订章节号、修订内容、批准人、批准日期 |
| 目录 | 清晰的章节及页码索引 |
| 颁布令 | 最高管理者签发颁布手册的通告和实施日期 |
| 公正性声明 | 机构对检验检测活动公正性的承诺 |
| 前言/机构概述 | 机构基本情况、手册的编制目的和适用范围 |
| 质量方针和目标 | 明确机构的质量方针、可量化的质量目标 |
| 组织与管理结构 | 组织架构图、各部门和人员的职责与权限 |
| 管理体系描述 | 体系范围、过程关系、文件控制程序 |
| 要素对应章节 | 按准则要素分章编写（目的范围、职责、原则描述、支持文件） |
| 附件 | 组织机构图、人员一览表、设备一览表、职能分配表等 |

### 2.3 质量方针和质量目标的制定

#### 2.3.1 质量方针

质量方针是由最高管理者正式发布的质量宗旨和方向，是全体职工的行为准则。质量方针应：
- 体现对质量管理的承诺
- 简单明确、便于记忆
- 不要形成标语口号或千篇一律的条文
- 除表述方针外，还要解释和说明方针的含义

#### 2.3.2 质量方针示例

检验检测机构质量方针示例："科学规范、优质高效、公平公正、服务客户"
- **科学规范**：所有检验检测活动必须建立在科学的基础上，严格按照标准、规程操作
- **优质高效**：在保证检验检测数据真实可靠的前提下，严格遵守工作时限，提高工作效率
- **公平公正**：平等对待一切客户，检验检测活动不受任何来自商业、行政或其他压力的影响
- **服务客户**：增强服务意识，满足客户的合理合法要求

#### 2.3.3 质量目标

质量目标应是可测量、可实现的，常见指标包括：
- 检测报告准确率达到98%以上
- 客户满意度达到95%以上
- 申诉投诉受理率100%
- 检测及时率不低于95%
- 设备校准/检定完成率100%
- 人员培训合格率100%

### 2.4 组织描述和管理架构

质量手册应明确：
- 组织架构图（内外部）
- 质量体系运行图
- 领导层、管理部门、一线部门的职能及相互关系
- 对检测工作质量有影响的人员（管理人员、操作人员、监督人员）的岗位职责、权限及任职条件

### 2.5 与标准条款的对应关系

质量手册的章节结构应尽量与准则的章节有对应关系，建议建立：
- "手册目录"与"条款对照表"双重验证
- 质量管理体系职责分配表（职能分配矩阵）
- 明确每个标准条款对应的负责部门和相关部门

## 三、程序文件编写

### 3.1 程序文件的定位

程序文件是质量手册的展开和细化，规定了各项质量活动的流程和方法，是实现质量管理目标的关键环节。程序文件更多涉及到工作步骤，希望通过对过程的约定，实现质量活动的可控。

### 3.2 程序文件的标准格式

程序文件应包含以下标准内容：

| 章节 | 内容说明 |
|------|----------|
| 目的 | 制定该程序的直接目的（控制过程）以及程序的目标 |
| 适用范围 | 界定活动的范围和活动的对象 |
| 职责 | 管理责任、实施责任、检查监督责任、记录归档责任 |
| 引用文件 | 列出程序所引用的相关法规、标准和其他文件 |
| 工作程序/控制要求 | 详细描述程序的工作流程、操作步骤和具体要求 |
| 支持性文件 | 引用的相关/支持性文件 |
| 质量记录 | 程序执行过程中需要产生的记录表格目录 |

## 四、作业指导书编写

### 4.1 作业指导书的定位

作业指导书（SOP，Standard Operating Procedure）也叫标准作业程序，是描述完成特定任务或操作的指导性文件。它跟程序文件没有本质的不同，只是更具体，可以细化到完成某个工作任务的"动作"。

### 4.2 作业指导书的类型

| 类型 | 适用范围 | 示例 |
|------|----------|------|
| 方法类 | 检测方法实施细则 | 珠宝玉石鉴定作业指导书、贵金属含量检测作业指导书 |
| 仪器类 | 设备操作规程 | 红外光谱仪操作规程、X射线荧光光谱仪操作规程 |
| 样品类 | 样品处理规程 | 样品接收与处理规程、留样管理规程 |
| 数据类 | 数据处理规程 | 检测数据修约规则、不确定度评定规程 |
| 消耗品类 | 试剂管理规程 | 标准溶液配制规程、化学试剂管理规程 |

### 4.3 作业指导书编写要点

1. **实操性**：针对具体检验项目，提供详细操作、维护、期间核查方法及安全指引
2. **全覆盖**：100%覆盖机构获批检验项目
3. **规范性**：需具备封面、刊头、修改控制页、正文等标准格式
4. **语言简洁**：尽可能采用量化表述，注重数据记录
5. **顺序合理**：按照检测流程顺序或标准条款顺序安排

### 4.4 设备操作规程编写要点

设备操作规程内容应包括：
- 设备名称、型号、编号
- 操作目的和适用范围
- 操作前检查（环境条件、设备状态）
- 操作步骤（开机、预热、校准、检测、关机）
- 日常维护保养要求
- 安全注意事项
- 应急处理措施
- 相关记录表格

## 五、记录表格设计

### 5.1 记录分类

检验检测机构的记录分为两大类：

| 类型 | 定义 | 示例 |
|------|------|------|
| 质量记录 | 管理体系活动中的过程和结果的记录 | 合同评审、分包控制、采购、内部审核、管理评审、纠正措施、预防措施和投诉 |
| 技术记录 | 进行检验检测活动的信息记录 | 原始观察、导出数据和建立审核路径有关信息的记录，检验检测、环境条件控制、人员、方法、设备管理、样品和质量监控等记录 |

### 5.2 原始记录设计要求

原始记录文件应为受控文件，一般设计为表格的形式。原始记录应包含以下信息：

1. **任务信息**：任务编号或样品编号（唯一性）
2. **检测标准（方法）**：所依据的标准编号和名称
3. **样品信息**：样品名称、唯一性编号、状态描述、接收日期
4. **检测环境**：温度、湿度等环境条件（对结果有影响时）
5. **仪器设备**：设备名称、唯一性编号、校准有效期、关键参数
6. **人员信息**：检测人员、复核人员签名
7. **检测数据**：原始观察结果、计算过程、公式
8. **时间信息**：检测日期、检测时间
9. **样品状态**：样品在检测前后的状态

Claim: 原始记录应包含充分的信息，确保该项检验检测在尽可能接近原始条件情况下能够重复

### 5.3 原始记录基本要求

| 要求 | 具体内容 |
|------|----------|
| 原始性 | 边实验边记录，不可漏记、抄记、事后补记或转抄 |
| 真实性 | 数据必须真实，表达应真实无误地反映测量仪器的输出 |
| 完整性 | 包含足够信息，以便识别不确定度的影响因素 |
| 可追溯性 | 能够追溯到样品、设备、人员、方法、环境等 |
| 有效性 | 使用的原始记录格式为有效的受控版本 |
| 修改规范 | 划改，不可擦涂，正确值填写在旁边，改动人签名 |

### 5.4 报告模板设计要求

## 六、文件控制要求

### 6.1 文件编制、审核、批准

| 文件类型 | 编制人 | 审核人 | 批准人 |
|----------|--------|--------|--------|
| 质量手册 | 质量负责人组织编制 | 管理者代表审核 | 最高管理者批准 |
| 程序文件 | 相关部门负责人组织编制 | 质量负责人/技术负责人审核 | 最高管理者/管理者代表批准 |
| 作业指导书 | 技术管理者组织编写 | 部门负责人审核 | 技术管理者批准 |
| 记录表格 | 使用部门编制 | 主管部门审核 | 质量负责人/技术负责人批准 |

### 6.2 文件编号规则

建立统一的文件编号系统，确保唯一性和可追溯性。推荐编号格式：

```
ABC-LAB-QP-001-A/0
|      |   |     |  |
|      |   |     |  └─ 修订次数
|      |   |     └─ 版本号
|      |   └─ 顺序号
|      └─ 文件类型(QM=手册, QP=程序, WI=指导书, QR=记录)
└─ 机构代号
```

### 6.3 文件控制全流程

文件控制归纳为"编、审、发、改、废、存、查"七个阶段：

```
文件编制 → 审核 → 批准 → 发放 → 使用 → 评审 → 修订/作废 → 归档
```

### 6.4 文件发放与回收

- 建立分发清单，明确文件发放的部门、岗位和份数
- 领用人需签收，确保责任到人
- 文件更新时，必须即时、全部回收旧版文件，防止误用
- 对回收的作废文件，加盖"作废留存"章存档或直接销毁并记录

### 6.5 文件修订与作废

- 文件经多次更改或需大幅度修改时，应进行换版
- 换版后原版文件作废，换发新版本
- 作废文件由文件管理人员回收
- 如因特殊原因保留的作废文件，应加盖"作废供参考"标识
- 作废文件销毁应填写销毁审批表，经批准后实施

Claim: 文件控制包括"编、审、发、改、废、存、查"七个阶段，确保各相关场所使用的文件均为有效版本

### 7.1 珠宝检测领域的特殊要求概述

珠宝检测实验室在建立管理体系时，除满足通用要求外，还需遵循CNAS-CL01-A015《检测和校准实验室能力认可准则在珠宝玉石、贵金属检测领域的应用说明》的特殊要求。

### 7.2 人员要求

| 要求项目 | 具体内容 |
|----------|----------|
| 人员固定性 | 所有从事珠宝玉石、贵金属检测的人员均应是长期固定工作人员，不允许在其他同类型实验室从事同类的检测活动 |
| 授权签字人 | 珠宝玉石鉴定分级实验室，每个检测场所至少应有2名授权签字人；贵金属检测实验室，每个检测场所至少有1名熟悉贵金属知识的专业人员 |
| 工作经历 | 大专毕业后从事珠宝领域检测工作8年以上；或本科毕业5年以上；或硕士以上3年以上 |
| 重新培训 | 两年以上未从事珠宝玉石检验的人员，应经过重新培训考核后才能上岗 |

## 八、体系文件编写流程

### 8.1 编写步骤

```
第1步：需求分析（1-2周）——对照标准确定文件需求
    ↓
第2步：框架设计（1-2周）——规划文件层级结构
    ↓
第3步：文件编写（1-2个月）——分模块编写，确保覆盖标准条款
    ↓
第4步：内部评审（1-2周）——组织内审，检查文件一致性和可操作性
    ↓
第5步：试运行与修订（3-6个月）——运行QMS，生成记录，优化文件
    ↓
第6步：提交准备——整理电子版，通过业务系统提交
```

### 8.2 编写注意事项

1. **标准符合性**：严格对照准则条款，避免遗漏
2. **版本控制**：文件编号清晰，记录修订历史
3. **简洁实用**：避免冗长，程序文件每份5-10页，作业指导书重点突出操作步骤
4. **可操作性**：与实际运行一致，确保手册与记录相符
5. **领域特定**：珠宝检测实验室应强调样品安保、分级细则、大型仪器操作等特殊要求
6. **语言规范**：使用专业术语，避免模糊表述

## 十、研究总结

### 10.1 关键发现

1. **体系文件四级架构**：检验检测机构管理体系文件采用质量手册、程序文件、作业指导书、记录表格四级金字塔结构，各层级之间需保持逻辑关联，下级文件支撑上级文件要求。

2. **质量手册核心地位**：质量手册是纲领性文件，应覆盖标准全部要素，建立条款对应表和职能分配表，确保每一项要求都有责任部门和执行文件。

3. **程序文件数量要求**：根据CMA和CNAS要求，检验检测机构通常需要建立25-30个程序文件，覆盖管理要求和技术要求全部要素。

4. **记录保存期限**：原始记录和报告保存期限不少于6年（依据《检验检测机构监督管理办法》第十三条），特殊领域另有规定的从其规定。

5. **电子文件管理趋势**：越来越多的实验室采用电子文件管理系统，需建立专门的电子文件控制程序，明确授权、权限、发布、修改、作废、备份和归档等要求。

6. **珠宝检测特殊要求**：
   - 人员要求严格：必须是长期固定工作人员，不得在其他同类型实验室兼职
   - 样品安保要求：需配备监控系统、保险柜等安保设施
   - 作业指导书要求：贵金属检测需覆盖抽样、检测流程、不确定度评估等；珠宝玉石鉴定需覆盖主要品种、定名规则和结果控制
   - 设备配置要求：需配备红外光谱仪等大型仪器，钻石分级需配备比色石
   - 数字化管理：深圳等地方已出台珠宝检测数字化管理规范

### 10.2 对珠宝检测实验室的建议

1. 建立符合CNAS-CL01-A015特殊要求的管理体系文件
2. 重点关注人员管理程序，确保满足"不得兼职"的严格要求
3. 完善样品安保相关程序，包括监控系统管理、保险柜管理等
4. 编制详细的检测作业指导书，覆盖所有获批检测项目
5. 建立设备校准和期间核查计划，确保比色石等关键设备的有效性
6. 关注数字化管理趋势，逐步推进检测过程的数字化管理
