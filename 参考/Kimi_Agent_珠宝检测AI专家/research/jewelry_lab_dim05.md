## 1. 概述与适用范围

### 1.1 标准基本信息

CNAS-CL01-A015:2018《检测和校准实验室能力认可准则在珠宝玉石、贵金属检测领域的应用说明》是由中国合格评定国家认可委员会（CNAS）制定的专项应用说明文件，结合珠宝玉石、贵金属检测的特点对CNAS-CL01:2018《检测和校准实验室能力认可准则》的部分条款要求作进一步说明。

```

Confidence: high
```

### 1.2 适用范围

本文件适用于珠宝玉石及贵金属检测领域的实验室认可。具体包括：
- **珠宝玉石检测**：对珠宝玉石的鉴定、分级、性能测试
- **贵金属检测**：用无损检测手段对贵金属材料、成品的含量/纯度、外观以及性能等进行的检测
- **贵金属化学分析**：应满足CNAS-CL01-A002《检测和校准实验室能力认可准则在化学检测领域的应用说明》的要求

```
Claim: 该领域涉及对珠宝玉石的鉴定、分级、性能测试，以及用无损检测手段对贵金属材料、成品的含量/纯度、外观以及性能等进行的检测。
Source: CNAS官方文件
URL: https://www.cnas.org.cn/cms_files/filemanager/328304128/attach/202412/5F260A438D15232018730F86025E5CEB.pdf
Date: 2018-03-01
Excerpt: "本文件适用于珠宝玉石及贵金属检测领域的实验室认可。该领域涉及对珠宝玉石的鉴定、分级、性能测试，以及用无损检测手段对贵金属材料、成品的含量/纯度、外观以及性能等进行的检测。"
Confidence: high
```

## 2. 环境控制要求

### 2.1 整体环境要求

珠宝检测实验室对环境条件有特殊要求，主要包括：

#### 2.1.1 色调与光照

```
Claim: 珠宝检测实验室的环境应为中性或灰色色调且无直射阳光；钻石分级应在色温5500~7200K范围内的钻石比色灯下进行。
Source: CNAS官方文件
URL: https://www.cnas.org.cn/cms_files/filemanager/328304128/attach/202412/5F260A438D15232018730F86025E5CEB.pdf
Date: 2018-03-01
Excerpt: "珠宝检测实验室的环境应为中性或灰色色调且无直射阳光；珠宝分级检测项目的光照条件应满足色温和显色指数要求。"
Confidence: high
```

#### 2.1.2 温湿度要求

```
Claim: 珠宝检测实验室推荐恒温恒湿条件为23℃±1℃，湿度50%±5%RH，防尘防光，需保持6个月环境日志，波动率<1%。
Source: 珠宝玉石贵金属检测实验室CNAS认证核查表攻略
URL: https://www.dkcnas.cn/jewelry-cnas-checklist-guide.html
Date: 2025-10-27
Excerpt: "场地配置：恒温恒湿室（23℃±1℃，湿度50%±5%RH），防尘防光。环境监控：实时传感器记录，温度波动<0.5℃。"
Confidence: medium
```

根据CNAS-CL01-A015:2018第6.3.3条款，如仪器设备和检测标准对检验环境有特殊要求，实验室应对**温度、湿度和电网电压**等进行监测和控制。

#### 2.1.3 环境监测要求

```
Claim: 实验室应对温度、湿度和电网电压等环境参数进行监测和控制，需保持环境记录6个月以上。
Source: CNAS-CL01-A015:2018
URL: https://www.cnas.org.cn/cms_files/filemanager/328304128/attach/202412/5F260A438D15232018730F86025E5CEB.pdf
Date: 2018-03-01
Excerpt: "如仪器设备和检测标准对检验环境有特殊要求，实验室应对温度、湿度和电网电压等进行监测和控制。"
Confidence: high
```

### 2.2 钻石分级专用光照条件

钻石分级对光源有特殊要求：

| 参数 | 要求范围 |
|------|----------|
| 色温 | 5500K ~ 7200K |
| 照度 | 1200lx ~ 2000lx |
| 显色指数 | ≥75 |
| 光源类型 | 连续谱荧光灯 |

```
Claim: 钻石颜色分级所用比色灯的色温范围为5500K~7200K，照度1200lx~2000lx，显色指数≥75。
Source: GB/T 16554-2003及比色灯技术规范
URL: https://baike.baidu.com/item/比色灯/22925488
Date: 2025-10-30
Excerpt: "依照国家标准设定为5500K至7200K范围，其中典型设备如GI-D60型分级灯的色温控制在5800K-6600K之间。显色指数需达到75以上。照度指标设定为1200lx至2000lx。"
Confidence: high
```

### 2.3 防振防尘要求

```
Claim: 珠宝检测实验室应采取防尘、防振措施，检测区需铺设软胶、地毯等防护材料，防止样品跌落受损。
Source: 珠宝首饰检验检测业务接待风险控制探讨
URL: http://www.knowcat.cn/n/20240710/798493.html
Date: 2024-07-10
Excerpt: "珠宝首饰检测实验室的地面和桌面通常均铺设软胶、地毯、地板革等，在检测各环节均应防止样品跌落受损。在样品传递过程中要防止相互碰撞。"
Confidence: high
```

#### 3.1.1 基本要求

```

Source: CNAS-CL01-A015:2018

Confidence: high
```

#### 3.1.2 授权签字人要求

```
Claim: 授权签字人应具有国家注册珠宝玉石质量检验师资格，并满足特定学历和工作年限要求。
Source: CNAS-CL01-A015:2018
URL: https://www.cnas.org.cn/images/rkgf/sysrk/rkyyzz/2016/12/05/3314D336147E16AE98CF4FB368CDA80F.pdf

Excerpt: "授权签字人应具有：a)大专毕业后，从事珠宝玉石、贵金属领域检测工作7年以上；或b)大学本科毕业，从事珠宝玉石、贵金属领域检测工作5年以上；或c)硕士学位以上(含)，从事珠宝玉石、贵金属领域检测工作3年以上；或d)非珠宝玉石鉴定、贵金属检测相关专业毕业人员，从事珠宝玉石、贵金属领域检测工作10年以上。"

```
