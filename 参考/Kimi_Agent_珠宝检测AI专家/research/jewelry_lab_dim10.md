## 一、评审类型

### 1.1 CMA资质认定评审类型

根据2023年版《检验检测机构资质认定评审准则》，CMA评审类型主要包括：

| 评审类型 | 适用情形 | 说明 |
|---------|---------|------|
| **首次评审** | 新申请资质认定的检验检测机构 | 对申请机构的技术能力和质量管理体系进行全面评审 |
| **复查换证评审** | 资质认定证书到期前申请延续 | 证书有效期6年，到期前需申请复查换证 |
| **扩项评审** | 已获证机构申请增加检测项目/参数 | 可单独申请，也可与监督/复评审合并进行 |
| **变更评审** | 机构信息、技术能力等发生变更 | 包括地址变更、法人变更、关键技术人员变更等 |
| **监督评审** | 获证后定期监督检查 | 通常每1-2年进行一次 |

Claim: CMA资质认定证书有效期为6年，在有效期内机构需每3年接受一次监督评审
Source: CMA资质认定全流程操作指南
URL: https://www.cmaqd.cn/article-item-88.html
Date: 2025-10-10
Excerpt: "整改通过后，省级市场监督管理部门会在20个工作日内颁发CMA资质认定证书，证书有效期为6年。在有效期内，机构需每3年接受一次监督评审"
Confidence: high

| 评审类型 | 适用情形 | 特点 |
|---------|---------|------|
| **初次评审** | 首次申请CNAS认可的实验室 | 全覆盖评审，包括全部要素和技术能力 |
| **监督评审** | 获认可后的定期监督 | 认可批准后12个月内进行首次监督，之后每2年一次 |

## 二、评审流程

### 2.1 CMA评审流程（2023版评审准则）

2023年12月1日起施行的《检验检测机构资质认定评审准则》建立了"三位一体"的评审程序体系：

**技术评审方式包括：**
1. **现场评审** - 适用于首次评审、扩项评审、复查换证评审（有实际能力变化时）、发生变更事项影响符合性的变更评审
2. **书面审查** - 适用于少量参数扩项或变更、上一许可周期无违法违规且申请事项无实质变化的复查换证
3. **远程评审** - 适用于因不可抗力无法前往现场、多地点相同检测活动等情形

Claim: 2023版评审准则建立了"三位一体"评审程序体系（现场评审、书面审查、远程评审），远程评审是重要创新点
Source: 2023新版《检验检测机构资质认定评审准则》及配套文件体系深度解析
URL: https://word.baidu.com/view/fdd8808c6c0203d8ce2f0066f5335a8102d26615.html
Date: 2025-11-01
Excerpt: "新版评审准则最显著的变化在于建立了'三位一体'的评审程序体系，包含了现场评审、书面审查和远程评审三种工作程序"
Confidence: high

### 2.2 CNAS评审流程

```
申请 → 受理 → 文件审查 → 现场评审 → 整改验证 → 评定批准 → 发证
```

**详细流程：**

| 阶段 | 主要内容 | 时间要求 |
|------|---------|---------|
| 申请与受理 | 通过LAS系统提交申请材料 | CNAS在3个月内安排现场评审 |
| 文件审查 | 评审组长审查质量手册、程序文件 | 文审通过后安排现场 |
| 现场评审 | 首次会议→现场参观→现场取证→沟通→末次会议 | 1-5天（视范围而定） |
| 整改验证 | 对不符合项进行纠正和纠正措施 | 一般2个月内完成 |
| 评定批准 | 评定委员会做出认可决定 | 整改验收后 |
| 发证 | 颁发认可证书及认可范围附件 | 证书有效期5年 |

Claim: CNAS一般情况下在受理申请后3个月内安排现场评审，整改期限一般为2个月
Source: CNAS实验室认可规则

Date: 2015-05-28
Excerpt: "CNAS在受理申请后,应在3个月内安排现场评审...纠正/纠正措施通常应在2个月内完成"
Confidence: high

### 2.3 现场评审程序

**标准现场评审程序（CMA和CNAS通用）：**

1. **预备会议**（评审组内部）：明确分工、评审计划
2. **首次会议**：评审组长主持，介绍评审目的、范围、日程安排
3. **现场观察/参观**：了解实验室环境、设施布局
4. **现场取证**：
   - 现场试验考核（盲样测试、人员比对、留样复测等）
   - 人员提问与座谈
   - 查阅原始记录和档案
   - 授权签字人考核
5. **评审组内部沟通**：汇总发现，确定不符合项
6. **与被评审方沟通**：反馈初步发现
7. **末次会议**：宣布评审结论，提出整改要求

Claim: 现场评审一般过程包括首次会议、现场参观、现场取证、评审组与申请方沟通、末次会议
Source: CNAS实验室认可评审工作指导书
URL: https://www.cnas.org.cn/cms_files/oldfile/images/sysrk/sysrkwjxz/sysrkpszlxz/2024/06/18/1718691397189047195.pdf
Date: 2024-06-18
Excerpt: "一般情况下,现场评审的过程是: a)首次会议; b)现场参观(需要时); c)现场取证; d)评审组与申请方沟通评审情况; e)末次会议"
Confidence: high

### 2.4 评审结论类型

**CMA评审结论（四种）：**

| 结论类型 | 含义 | 后续处理 |
|---------|------|---------|
| **符合** | 管理体系符合评审准则并有效运行 | 可直接建议批准资质认定 |
| **基本符合** | 体系基本符合，发现不符合项 | 完成整改并经文件审查后报送 |
| **基本符合（需现场复核）** | 不符合项需现场复核确认 | 整改后接受现场复核 |
| **不符合** | 管理体系多处不符合或能力差距大 | 不能推荐资质认定 |

**CNAS评审结论（三种）：**

| 结论类型 | 含义 |
|---------|------|
| **符合** | 满足认可要求 |
| **基本符合** | 存在不符合项，需整改跟踪 |
| **不符合** | 不满足认可要求 |

Claim: CMA现场评审结论分为符合、基本符合、基本符合（需现场复核）、不符合四种
Source: 江苏省检验检测机构资质认定评审员晋级培训考核资料汇编
URL: https://scjgj.jiangsu.gov.cn/attach/0/c3b7d8bec2f846268beb0d98b726e086.pdf
Date: 2024-09-01
Excerpt: "评审结论分为符合、基本符合、基本符合(需现场复核)、不符合四种"
Confidence: high

## 三、迎审准备

### 3.1 资料准备清单

**管理体系文件：**
- 质量手册（覆盖全部要素）
- 程序文件（≥21个典型程序）
- 作业指导书/检测细则
- 记录表格

**运行记录（至少3-6个月）：**
- 内部审核报告
- 管理评审报告
- 人员培训记录
- 设备校准/维护/期间核查记录
- 质量控制记录
- 样品管理记录
- 投诉处理记录
- 不符合项及纠正措施记录

**技术文件：**
- 检测能力附表
- 方法验证/确认报告
- 测量不确定度评定报告
- 典型检测报告及原始记录

Claim: 迎审准备清单涵盖"硬件+软件"基础，管理体系运行≥3个月，内审覆盖率100%
Source: CMA现场评审高分通过实战攻略

Confidence: high

### 3.2 人员准备

**关键岗位人员准备：**
- 技术负责人：掌握方法验证、不确定度评定
- 质量负责人：熟悉内审、管理评审流程
- 授权签字人：准备7方面考核内容（见第四节）
- 检测人员：熟练操作、熟悉SOP
- 内审员：持证上岗

**人员准备要点：**
1. 全员培训体系文件，确保"说写做一致"
2. 关键岗位人员提前演练问答
3. 组建迎审小组（质量+技术+行政）
4. 安排熟悉实验室的陪同人员

### 3.3 设备准备

- 设备台账完整，校准证书在有效期内
- 期间核查计划及记录完整
- 设备状态标识清晰（合格/停用/限用）
- "一机一档"管理到位
- 备用设备准备（关键项目）

### 3.4 现场准备

- 实验室环境整洁，功能分区明确
- 环境监控记录完整（温湿度等）
- 样品流转路径清晰，标识到位
- 安全防护设施到位
- 文件分类归档，便于快速调取

### 3.5 模拟评审

**模拟评审的重要性：**

模拟评审（预评审）是正式评审前的"压力测试"，建议邀请外部专家（如CNAS主任评审员）进行。

**模拟评审内容：**
- 完全仿照正式评审流程
- 抽查文件、现场观察操作、访谈人员
- 重点检查"说写做"一致性
- 发现问题立即整改

Claim: 模拟评审是查漏补缺最有效的手段，可暴露体系运行、设备管理、人员操作、记录填写等方面的真实问题

## 四、现场评审要点

### 4.1 首次会议

**首次会议内容：**
- 介绍评审组成员及分工
- 明确评审目的、依据、范围
- 实验室负责人介绍概况
- 确认评审日程安排
- 宣布公正性、保密声明
- 确定陪同人员和工作场所

**应对要点：**
- 准备5-10分钟PPT概述机构概况
- 明确评审范围，确认需配合人员
- 有疑问当场提出
- 建立良好沟通基调

### 4.2 现场观察

评审员重点关注：
- 实验室环境条件（温湿度、洁净度等）
- 区域隔离和标识管理
- 安全防护设施
- 样品流转路径（防止交叉污染）
- 设备布局和状态标识

### 4.3 现场试验

**现场试验方式：**

| 试验方式 | 说明 |
|---------|------|
| **操作演示** | 检测人员按SOP进行现场操作 |
| **盲样考核** | 对已知结果的样品进行检测 |
| **人员比对** | 不同人员检测同一样品 |
| **留样复测** | 对保留样品重新检测 |
| **仪器比对** | 不同仪器检测同一样品 |

**应对要点：**
- 严格按照作业指导书操作
- 如实记录原始数据
- 确保数据可复现、可追溯
- 操作人员熟悉方法原理和注意事项

Claim: 现场试验方式包括操作演示、盲样考核、比对试验（人员/仪器），验证重点为检验方法有效性、数据准确性、操作合规性和记录完整性
Source: CMA资质认定流程

Confidence: high

### 4.4 授权签字人考核

**授权签字人考核7个方面（CMA要求）：**

1. 具有相应职责和权利，对检测结果的完整性和准确性负责
2. 与检测技术接触紧密，掌握检测项目限制范围
3. 熟悉检验检测标准、方法及规程
4. 有能力对检测结果进行评定，了解不确定度
5. 了解设备维护保养及校准规定，掌握校准状态
6. 熟悉记录、报告及其核查程序
7. 了解认定条件、机构义务等有关规定

**考核方式：**
- 面对面提问，评审组长主持
- 填写《检验检测机构现场考核授权签字人评价记录表》
- 考核时长约15-30分钟

**常见问题类别：**
- 工作经历相关
- 职责权利相关
- 管理体系认知
- 技术标准方法掌握
- 设备状态了解
- 结果评价判断能力

Claim: 授权签字人必须考核7个方面，考核方式面对面进行，对每个授权签字人填写一张评价记录表

Confidence: high

### 4.5 末次会议

**末次会议内容：**
- 报告评审情况
- 说明发现的主要问题
- 确认不符合项
- 宣布评审结论
- 提出整改要求和时限
- 说明评审的局限性

**应对要点：**
- 认真听取评审意见
- 对不符合项确认事实
- 记录整改要求和时间节点
- 避免误解导致整改偏差

## 五、常见问题及不符合项分析

### 5.1 不符合项分级

**CNAS不符合项分级（依据CNAS-GL008）：**

| 级别 | 定义 | 典型示例 |
|------|------|---------|
| **严重不符合项** | 影响实验室诚信或显著影响技术能力、结果准确性和管理体系有效运作 | 提交虚假申请资料；不做试验直接出报告；人员能力明显不足；缺乏关键设备；管理体系某些环节失效 |
| **一般不符合项** | 偶发的、独立的对结果和管理体系没有严重影响的不符合 | 设备未按期校准；试剂过期；记录不完整；纠正措施未验证 |

**CMA不符合项分级：**

| 级别 | 说明 |
|------|------|
| **严重不符合项** | 影响结果可靠性或体系运行的关键问题 |
| **一般不符合项** | 对结果影响较小或局部问题 |

Claim: CNAS将不符合项分为严重不符合项和一般不符合项，严重不符合项影响实验室诚信或显著影响技术能力，一般不符合项为偶发的独立问题
Source: CNAS-GL008《实验室认可评审不符合项分级指南》
URL: https://www.cnas.org.cn/api-gateway/jpaas-web-server/front/document/download
Date: 2018
Excerpt: "严重不符合项：影响实验室诚信或显著影响技术能力、检测或校准结果准确性和可靠性...一般不符合项：偶发的、独立的对检测或校准结果没有严重影响的不符合项"
Confidence: high

### 5.2 人员方面常见问题

## 六、整改要求

### 6.1 整改时限

**CMA整改时限：**
- 一般不超过**30个工作日**
- 复杂问题可申请延期（通常不超过30个工作日）
- 逾期未完成的，相应评审项目判定为不合格

**CNAS整改时限：**
- 初次评审和扩项评审：**2个月**
- 监督评审和复评审：**2个月**
- 涉及技术能力的不符合（监督/复评）：**1个月**
- 严重不符合项：**1个月**

### 6.2 整改报告编写

**整改报告编写原则：**
1. **针对性** - 严格对应每一个不符合项
2. **根本性** - 分析根本原因，制定彻底解决问题的措施
3. **证据性** - 附上清晰、有说服力的证明材料
4. **及时性** - 在规定期限内完成并提交

**整改报告内容框架：**
```
1. 封面/标题页
2. 引言（评审基本信息、不符合项数量）
3. 不符合项逐条整改
   - 不符合项描述
   - 原因分析
   - 纠正措施
   - 实施情况
   - 证明材料（附件）
4. 总结
5. 附件（证据材料）
```

### 6.3 整改证据

**常见整改证据类型：**
- 修订后的文件（含审批记录）
- 培训记录（签到表、课件、考核记录）
- 校准/检定证书
- 期间核查记录
- 质量控制记录
- 照片/截图
- 会议纪要

- 声明不在两个以上机构从业

**策略四：设备管理到位**
- 按时申报定期校准/检定
- 保留完整有效的证书
- 建立设备管理台账

**策略五：报告管理合规**
- 确保报告和原始记录完整
- 数据真实、可追溯
- 审核流程规范

### 8.5 授权签字人能力验证考核

部分地区在双随机检查中增加了授权签字人集中考核环节：
- 选取重点领域
- 分区域分领域组织
- 验证授权签字人能力持续维持情况

## 九、研究发现汇总

### 9.1 关键发现

1. **评审类型多样**：CMA和CNAS各有5种以上评审类型，实验室应根据自身情况准确选择申请类型

2. **2023版CMA评审准则重大变化**：建立了现场评审、书面审查、远程评审"三位一体"体系，删除了对管理体系部分程序的强制要求，更强调条件的符合性

3. **珠宝检测实验室有特殊要求**：
   - 人员方面：CGC资格是硬性要求，授权签字人必须具备CGC资格
   - 设备方面：比色石需溯源至国家标准，有数量要求
   - 环境方面：钻石分级有严格的色温和环境色调要求
   - 质控方面：人为定级项目每年至少2次内部质控

4. **整改时限明确**：
   - CMA：30个工作日
   - CNAS：一般2个月，技术能力问题1个月

5. **不符合项分级管理**：
   - 严重不符合项：影响诚信或技术能力，需立即整改
   - 一般不符合项：局部问题，限期整改

6. **双随机监管趋严**：珠宝玉石检测被列为重点监管领域，需高度重视合规经营

### 9.2 迎审核心建议

1. **提前准备**：至少提前1-3个月启动迎审准备工作
2. **文件体系**：确保"全、准、实"，记录可追溯
3. **模拟评审**：邀请外部专家进行模拟评审是最有效的查漏补缺手段
4. **人员培训**：确保全员理解体系文件，"说写做一致"
5. **技术能力**：重点保障人、机、料、法、环五要素齐备
6. **沟通态度**：谦虚配合，用数据说话，避免争辩

### 9.3 评审高频不符合项统计

| 问题类别 | 占比估计 |
|---------|---------|
| 记录类问题 | ~30% |
| 设备管理问题 | ~20-25% |
| 文件控制问题 | ~15-20% |
| 人员能力问题 | ~15% |
| 质控问题 | ~10-15% |
| 其他 | ~5-10% |
