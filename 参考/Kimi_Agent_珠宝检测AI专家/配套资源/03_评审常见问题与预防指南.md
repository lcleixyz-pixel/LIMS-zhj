## 第一部分 CMA评审常见问题（按类别）

### A. 人员方面（10项）

> 📋 **对应标准条款**：RB/T 214-2017 第4.2条；CNAS-CL01:2018 第6.2条

| 序号 | 不符合项描述 | 对应条款 | 预防措施 |
|:---:|---|---|---|
| A-01 | 质量负责人/技术负责人/授权签字人任职文件缺失或职责规定不明确 | RB/T 214-2017 4.2.3/4.2.4 | ①制定正式的任命文件，由最高管理者签发；②文件中明确岗位职责、任职条件、代理关系；③授权签字人需经资质认定部门考核批准 |
| A-02 | 人员档案不完整，缺少学历证明、职称证书、培训记录等 | RB/T 214-2017 4.2.6/4.2.7 | ①建立《人员技术档案清单》；②入职时收集全部资质证明原件并复印归档；③每年更新档案，补充培训记录、继续教育证明 |
| A-03 | 关键岗位人员兼职未声明或兼职机构数量超过规定 | RB/T 214-2017 4.2.3 | ①所有兼职人员在《人员情况表》中声明；②记录兼职单位的名称、职务、兼职时间；③质量负责人不得兼职其他检验检测机构 |
| A-04 | 授权签字人考核记录缺失或考核内容与授权领域不匹配 | RB/T 214-2017 4.2.4 | ①建立授权签字人考核程序；②考核覆盖申请签字的全部领域；③保留考核试卷/记录，保存至授权取消后6年 |
| A-05 | 年度培训计划未实施或培训记录不完整（缺签到表、教材、考核记录） | RB/T 214-2017 4.2.6 | ①年初制定年度培训计划并经审批；②培训实施时保留：通知、签到表、教材/课件、照片、考核记录；③培训效果进行评价 |
| A-06 | 检测人员无证上岗或上岗考核记录缺失 | RB/T 214-2017 4.2.5/4.2.6 | ①制定《人员上岗考核程序》；②检测人员须经理论+实操考核合格后持证上岗；③上岗证应注明授权检测项目/方法 |
| A-07 | 监督员任命文件缺失或监督活动未有效开展 | RB/T 214-2017 4.2.7 | ①任命经培训合格的监督员（比例建议1:5~1:8）；②制定年度监督计划并实施；③监督记录应包括：被监督人、监督内容、发现问题、改进建议 |
| A-08 | 人员能力监控未实施或记录不符合要求 | CNAS-CL01:2018 6.2.5 | ①建立人员能力监控程序；②采用盲样测试、人员比对、留样复测、现场观察等方式；③对新进人员、关键岗位人员增加监控频次 |
| A-09 | 关键岗位人员流动未办理交接手续，影响检测工作连续性 | RB/T 214-2017 4.2.3 | ①建立人员变动管理程序；②关键岗位人员变动需提前1个月报告；③办理工作交接清单，交接记录存档 |
| A-10 | 培训内容与实际工作需求脱节，针对性不强 | RB/T 214-2017 4.2.6 | ①培训需求调查每年至少一次；②根据能力评估差距确定培训内容；③珠宝专业培训应覆盖：仪器操作、标准更新、宝石鉴定新技术等 |

⚠️ **重点关注**：授权签字人是评审必查项，务必确保：有任命文件→经考核合格→在资质认定证书附表中明确→考核记录完整→仅在其授权领域内签发报告。

### B. 设备设施方面（10项）

> 📋 **对应标准条款**：RB/T 214-2017 第4.4条；CNAS-CL01:2018 第6.4条

| 序号 | 不符合项描述 | 对应条款 | 预防措施 |
|:---:|---|---|---|
| B-01 | 设备校准证书过期或校准计划未按时执行 | RB/T 214-2017 4.4.3 | ①建立《设备校准计划表》，按证书有效期倒排；②设置证书到期前3个月预警；③校准机构需具备CNAS认可或法定计量授权资质 |
| B-02 | 设备期间核查未实施或核查方法/频次不合理 | RB/T 214-2017 4.4.3; CNAS-CL01-G004 | ①制定《期间核查计划》，明确核查对象、方法、频次、判定准则；②对使用频繁、易漂移的设备（如电子天平、分光光度计）增加核查频次；③核查记录应包括：标准物质/参考值、核查结果、判定结论 |
| B-03 | 标准物质/标准样品未溯源或证书过期 | RB/T 214-2017 4.4.6 | ①建立标准物质台账，标注证书编号、有效期、不确定度；②优先选用有证标准物质（CRM）；③标准溶液配制后需标定并标注有效期 |
| B-04 | 环境条件监控记录缺失或不完整（温湿度、光照等） | RB/T 214-2017 4.4.4 | ①各功能区域配置温湿度自动记录仪；②制定监控点分布图和监控频次要求；③记录超出控制范围时的处置措施 |
| B-05 | 钻石比色灯光源色温不符合要求（应为5500-7200K） | CNAS-CL01-GXXX | ①配备标准钻石比色灯（D65光源，色温5500-7200K）；②定期检测光源色温并记录；③比色灯管按使用时长定期更换 |
| B-06 | 电子天平放置位置不当（振动、气流、阳光直射） | RB/T 214-2017 4.4.4 | ①天平台应独立基础、防振；②远离门窗、空调出风口、人员频繁走动区域；③使用水准仪调平，每次使用前检查水平 |
| B-07 | 设备档案不完整，缺验收记录、使用记录、维护记录 | RB/T 214-2017 4.4.4 | ①一机一档，档案内容：申购单、验收记录、校准证书、使用记录、维护记录、维修记录、期间核查记录；②使用记录应包含：样品编号、检测项目、使用人、使用时间 |
| B-08 | 设备状态标识管理混乱（合格/准用/停用标识不清） | RB/T 214-2017 4.4.3 | ①三色标识管理：绿色（合格）、黄色（准用/限用）、红色（停用）；②标识应包含：设备编号、校准日期、有效期、校准机构；③停用设备应隔离存放 |
| B-09 | 危险废物（酸液、有机溶剂）储存设施不符合安全要求 | RB/T 214-2017 4.4.4 | ①设置独立危化品储存柜（防火、防泄漏、通风）；②废液分类收集，标识清晰；③与有资质单位签订危废处置协议 |
| B-10 | 安全防护设施（洗眼器、通风橱、灭火器）未定期检查 | RB/T 214-2017 4.4.4 | ①建立安全设施检查清单；②每月检查一次并记录；③应急设施（洗眼器、紧急喷淋）每季度试水验证 |

⚠️ **重点关注**：设备校准是评审高频问题项。特别注意：校准证书上的设备编号应与实验室内部编号一致；校准结果应进行确认并形成文件；设备修正因子/修正值应正确使用并更新。

### C. 方法验证方面（10项）

> 📋 **对应标准条款**：RB/T 214-2017 第4.5.14条；CNAS-CL01:2018 第7.2条

| 序号 | 不符合项描述 | 对应条款 | 预防措施 |
|:---:|---|---|---|
| C-01 | 方法验证不完整，缺少精密度、准确度、检出限等关键指标 | RB/T 214-2017 4.5.14 | ①建立《方法验证/确认程序》；②验证内容至少包括：精密度、准确度（回收率或与标准值比对）、检出限/定量限、线性范围、选择性；③形成验证报告并经审批 |
| C-02 | 标准查新不及时，使用作废标准进行检测 | RB/T 214-2017 4.5.14 | ①指定专人负责标准查新，每月查询国家标准化管理委员会网站；②建立《标准受控清单》，标注发布日期、实施日期、作废日期；③新标准实施前完成方法验证和人员培训 |
| C-03 | 非标方法（如自编方法、客户指定方法）未进行方法确认 | RB/T 214-2017 4.5.14; CNAS-CL01 7.2.2 | ①非标方法使用前须进行方法确认；②确认内容比方法验证更全面，包括：抗干扰性、稳健性、不同操作者的适用性；③非标方法使用需经客户同意并在报告中声明 |
| C-04 | 测量不确定度未评定或评定不合理（评定模型错误、分量遗漏） | RB/T 214-2017 4.5.15; CNAS-CL01-G003 | ①建立《测量不确定度评定程序》；②识别不确定度来源：取样、样品处理、仪器校准、环境条件、人员操作、方法偏差等；③使用GUM法或蒙特卡洛法评定，保留计算过程 |
| C-05 | 方法验证报告缺少原始数据支持或数据统计方法错误 | RB/T 214-2017 4.5.14 | ①保留完整的验证原始数据（色谱图、光谱图、称量记录等）；②使用正确的统计方法（Grubbs检验异常值、t检验显著性等）；③验证样品应有代表性，覆盖常规浓度范围 |
| C-06 | 检测实施细则（SOP）内容不完整或与标准方法不一致 | RB/T 214-2017 4.5.14 | ①SOP应包含：目的、适用范围、原理、试剂/设备、操作步骤、结果计算、质量控制、安全注意事项；②SOP须经过审核批准；③偏离标准方法时应经技术判断和客户同意 |
| C-07 | 方法变更后未重新进行验证或未及时通知客户 | RB/T 214-2017 4.5.14 | ①建立方法变更控制程序；②变更后重新进行验证并形成记录；③如有必要，书面通知已受影响的客户 |
| C-08 | 检测方法选择性未经验证，存在干扰物质影响 | CNAS-CL01 7.2.1 | ①评估样品基质中可能存在的干扰物质；②进行加标回收试验验证选择性；③对珠宝检测中常见的共存元素干扰进行识别和消除 |
| C-09 | 校准曲线线性范围未经验证或相关系数要求不明确 | CNAS-CL01 7.2.1 | ①校准曲线应覆盖待测样品的浓度范围；②相关系数r应满足方法要求（通常r≥0.999）；③每批样品应重新建立校准曲线 |
| C-10 | 方法检出限（MDL）和定量限（LOQ）确定方法不正确 | CNAS-CL01-G004 | ①采用空白标准偏差法或信噪比法；②MDL=3.14×Sbl（n≥7），LOQ=10×Sbl；③定期验证检出限（每季度或仪器维修后） |

⚠️ **重点关注**：方法验证是技术评审的核心内容。珠宝检测实验室常见的方法包括：贵金属含量测定（GB/T 9288、GB/T 21198等）、钻石分级（GB/T 16554）、珠宝玉石鉴定（GB/T 16552、GB/T 16553），每类方法均需独立验证。

### D. 样品管理方面（10项）

> 📋 **对应标准条款**：RB/T 214-2017 第4.5.18条/4.5.20条；CNAS-CL01:2018 第7.4条

| 序号 | 不符合项描述 | 对应条款 | 预防措施 |
|:---:|---|---|---|
| D-01 | 样品标识不清晰或不唯一，无法追溯 | RB/T 214-2017 4.5.18 | ①建立唯一性标识规则（编号规则：年月+类别+流水号）；②标识应牢固、清晰，不易脱落或模糊；③分样后的子样应保留与原样的关联标识 |
| D-02 | 样品流转记录不完整，缺少交接签字或时间记录 | RB/T 214-2017 4.5.20 | ①使用《样品流转单》全程记录：接收→登记→分发→检测→审核→出具报告→留样/退还；②各环节交接须签字确认并记录时间 |
| D-03 | 高价值样品（大颗粒钻石、高档翡翠等）安保措施不足 | RB/T 214-2017 4.4.4 | ①设置保险柜或专用保管箱；②实行双人双锁管理；③安装监控摄像头，覆盖样品收发、检测全过程；④高价值样品检测须全程在监控下进行 |
| D-04 | 样品保存条件不符合要求（温湿度、光照、防化学腐蚀） | RB/T 214-2017 4.5.18 | ①不同类别样品分区存放：贵金属、有机宝石（珍珠、珊瑚）、玉石、钻石；②有机宝石需控制湿度（50%-70%RH），避免干燥开裂；③翡翠等玉石避免强光直射 |
| D-05 | 样品在待检/在检/已检区域未分开存放 | RB/T 214-2017 4.5.18 | ①设置明确的物理分区或标识：待检区（黄色）、在检区（蓝色）、已检区（绿色）；②防止样品混淆和交叉污染 |
| D-06 | 留样样品保存期限不足或留样量不够复测 | RB/T 214-2017 4.5.18 | ①制定留样规定：一般样品留样3个月，争议样品留样6个月至纠纷解决；②留样量应满足至少一次完整复测；③留样条件应能保证样品性质稳定 |
| D-07 | 样品接收时未充分与客户沟通检测需求和样品特性 | RB/T 214-2017 4.5.4 | ①制定《样品接收检查表》：样品状态描述、特殊性质声明、客户检测要求、是否需要破坏性检测的确认；②贵重物品拍照留存外观状态 |
| D-08 | 退样手续不完整，缺少客户签收记录 | RB/T 214-2017 4.5.18 | ①退样时填写《样品退还单》，记录退样时间、退还人、接收人签字；②贵重样品建议采用保价快递或客户自取方式，保留快递单据 |
| D-09 | 样品在检测过程中发生损坏未记录和处理 | RB/T 214-2017 4.5.18 | ①制定样品异常处理程序；②检测前拍照记录样品状态；③发生损坏时立即记录，报告技术负责人，书面通知客户并协商处理方案 |
| D-10 | 样品编号与检测报告编号无关联，追溯困难 | RB/T 214-2017 4.5.27 | ①建立编号关联规则：检测报告编号包含或关联样品编号；②信息系统中实现一键追溯：报告→样品→检测记录→设备→人员→环境 |

⚠️ **重点关注**：珠宝样品价值高、易损坏、检测过程可能涉及破坏性方法（如密度测试），务必在样品接收时与客户充分确认检测方法，并告知可能的风险。

### E. 质量控制方面（10项）

> 📋 **对应标准条款**：RB/T 214-2017 第4.5.19条；CNAS-CL01:2018 第7.7条；CNAS-CL01-G002

| 序号 | 不符合项描述 | 对应条款 | 预防措施 |
|:---:|---|---|---|
| E-01 | 年度质控计划未覆盖全部检测项目/方法/人员 | RB/T 214-2017 4.5.19 | ①每年初制定《年度质量控制计划》；②覆盖原则：每个项目、每个方法、每个检测人员至少参加一次质控活动；③质控方式：人员比对、设备比对、留样复测、加标回收、能力验证、实验室间比对 |
| E-02 | 能力验证未参加或结果不满意（|z|>2）未采取纠正措施 | RB/T 214-2017 4.5.19; CNAS-CL01-G002 | ①查询CNAS/Nordic/中检院等PTP发布的能力验证计划；②优先参加CNAS认可PTP组织的能力验证；③结果不满意时：立即停止相关检测→分析原因→采取纠正措施→验证有效性→恢复检测 |
| E-03 | 人员比对未实施或比对结果评价方法不正确 | RB/T 214-2017 4.5.19 | ①同一检测项目至少2名人员参加比对；②评价方法：En值法（En≤1满意）或t检验法；③比对结果有异议时查找原因（人员操作、设备状态、环境等） |
| E-04 | 设备比对未实施或设备间差异超出可接受范围 | RB/T 214-2017 4.5.19 | ①多台设备检测同一项目时应安排设备比对；②使用同一样品在相同条件下进行比对；③评价标准：两台设备结果的差异应≤扩展不确定度或方法规定的允许差 |
| E-05 | 质控图未绘制或超出控制限未进行处理 | CNAS-CL01 7.7.1 | ①对稳定检测项目绘制Levey-Jennings质控图；②控制限设置：中心线（均值）、警告线（±2s）、控制限（±3s）；③超出警告线时增加关注，超出控制限时立即停止检测并查找原因 |
| E-06 | 加标回收率未在方法要求范围内，未分析原因 | RB/T 214-2017 4.5.19 | ①每批样品进行加标回收试验；②回收率控制范围：一般90%-110%（视方法要求而定）；③回收率异常时：检查称量、稀释、仪器状态、基质干扰等 |
| E-07 | 质量控制结果未纳入管理评审输入 | RB/T 214-2017 4.5.19/4.5.13 | ①汇总年度质量控制结果，分析趋势；②将质控数据作为管理评审的输入之一；③根据质控结果调整质控计划或采取改进措施 |
| E-08 | 留样复测频次不足或结果差异未分析原因 | RB/T 214-2017 4.5.19 | ①每季度至少进行一次留样复测；②复测人员应为原检测人员以外的人员；③两次结果差异应≤扩展不确定度或方法允许差 |
| E-09 | 质控记录与检测记录未关联，无法对应查询 | CNAS-CL01 7.7.1 | ①质控样品与常规样品同步检测并在同一记录中体现；②记录中标注质控类型和判定结果；③信息系统应能关联质控数据和检测数据 |
| E-10 | 质控样品与待测样品基质差异大，质控无实际意义 | CNAS-CL01 7.7.1 | ①质控样品应与待测样品基质相似；②使用有证标准物质时选择与样品基质匹配的CRM；③自制质控样品时应模拟实际样品组成 |

⚠️ **重点关注**：CNAS要求参加能力验证的领域须满足CNAS-AL07《能力验证领域和频次表》要求。珠宝检测实验室重点关注：贵金属含量测定、钻石分级、宝石鉴定等领域的能力验证活动。

## 第二部分 不符合项整改要点

### 一、不符合项整改报告标准格式

📋 **整改报告应包含以下内容**：

| 章节 | 内容要求 |
|---|---|
| 1. 不符合项描述 | 复述评审组开具的不符合项内容（照抄原文，不得修改） |
| 2. 不符合原因分析 | 采用5Why分析法或鱼骨图法，深入查找根本原因（不只停留在表面原因） |
| 3. 纠正措施 | 针对根本原因采取的系统性整改措施，应具有可操作性 |
| 4. 纠正（立即行动） | 已经采取的即时纠正行为 |
| 5. 预防措施 | 防止类似问题再次发生的措施 |
| 6. 整改证据 | 附上相关证据材料（培训记录、修订文件、照片、检测报告等） |
| 7. 整改完成日期 | 明确填写整改完成的具体日期 |
| 8. 责任人签字 | 整改责任人、部门负责人、质量负责人签字确认 |

### 二、纠正措施与根本原因分析

#### 2.1 根本原因分析方法

```
┌─────────────────────────────────────────────────────────┐
│                    鱼骨图分析法                           │
├─────────────┬─────────────┬─────────────┬───────────────┤
│   人（Man）  │ 机（Machine）│ 料（Material）│ 法（Method）  │
│             │             │             │               │
│ • 人员能力不足│ • 设备老化   │ • 样品不具代表性│ • 方法选择不当 │
│ • 培训不到位 │ • 校准过期   │ • 标准物质失效 │ • SOP不完善   │
│ • 责任心不强 │ • 维护不足   │ • 试剂过期     │ • 操作步骤不清 │
│ • 工作量大   │ • 精度不够   │ • 环境不达标   │ • 记录表格缺陷 │
├─────────────┴─────────────┴─────────────┴───────────────┤
│                    环（Environment）                      │
│  • 温湿度失控  • 照明不足  • 电磁干扰  • 振动干扰        │
└─────────────────────────────────────────────────────────┘
```

#### 2.2 纠正措施制定原则

| 原则 | 说明 |
|---|---|
| 针对性 | 措施直接针对根本原因，而非表面现象 |
| 可操作性 | 措施明确具体，可执行、可检查、可验证 |
| 系统性 | 考虑对整体体系的影响，避免顾此失彼 |
| 时限性 | 明确完成时限，确保及时关闭不符合项 |
| 有效性 | 通过后续跟踪验证措施是否真正有效 |

#### 2.3 常见不符合项的纠正措施示例

| 不符合项 | 根本原因 | 纠正措施 | 验证方式 |
|---|---|---|---|
| 设备校准证书过期 | 证书管理无预警机制 | ①建立校准证书到期预警台账（提前3个月）；②设置电子日历提醒；③指定专人管理 | 查看预警台账和后续证书 |
| 人员培训记录缺失 | 培训管理程序执行不严 | ①修订《培训管理程序》，明确记录要求；②补充相关培训；③增加培训效果评价环节 | 查看修订后程序和培训记录 |
| 质控图超出控制限未处理 | 质控管理意识不足 | ①组织质控管理专项培训；②修订质控程序，明确超限处理流程；③补充超限原因分析记录 | 查看培训记录和修订程序 |
| 标准使用作废版本 | 标准查新机制缺失 | ①建立月度标准查新制度；②订阅标准更新推送服务；③新标准实施前完成培训和验证 | 查看查新记录和受控清单 |

| 证据类型 | 具体内容 | 适用场景 |
|---|---|---|

### 二、主观判断项目的质控要求

珠宝检测中大量依赖检测人员的经验和主观判断，这类项目的质量控制尤为重要。

#### 2.1 主观判断项目清单

| 检测项目 | 主观判断内容 | 质控要求 |
|---|---|---|
| **钻石颜色分级** | 比色判断颜色级别 | ①定期用标准比色石校准；②2人独立比色，结果差异>1级时第3人仲裁；③参加能力验证/测量审核 |
| **钻石净度分级** | 内含物观察与级别判定 | ①多人复核制度；②定期用标准净度标样进行一致性检验；③记录典型内含物特征 |
| **翡翠颜色评价** | 颜色色调、明度、饱和度 | ①建立标准比色板/比色卡；②多人评价取一致结果；③拍照留存比对 |
| **珍珠光泽/颜色评价** | 光泽强弱、颜色均匀度 | ①与标准样品比对；②多人员独立评价；③环境光源标准化 |
| **祖母绿净度优化评价** | 充填程度判断 | ①显微镜下多放大倍数观察；②与标准图谱比对；③记录充填物类型和分布 |
| **宝石切工评价** | 对称性、抛光质量 | ①使用切工比例仪量化测量；②结合目视评价；③参照标准切工图谱 |

#### 2.2 主观判断项目质控措施

| 质控方式 | 实施要求 | 频次 |
|---|---|---|
| **人员间比对** | 相同样品由不同检测人员独立判断，评价一致性 | 每月至少1次 |
| **标准样品校准** | 使用有证标准样品/标准图谱进行定期校准 | 每季度至少1次 |
| **盲样测试** | 技术负责人/质量监督员插入盲样进行考核 | 每季度至少1次 |
| **能力验证** | 参加权威机构组织的珠宝检测能力验证 | 每年至少1次 |
| **测量审核** | 送样至国家级珠宝质检中心进行比对 | 每年至少1次 |
| **视频记录** | 对主观判断过程进行视频记录备查 | 重要样品必录 |

⚠️ **特别关注**：对于钻石分级等高度依赖主观判断的项目，CNAS评审时可能要求现场进行盲样测试，检测人员应做好充分准备。

### 三、贵金属检测的方法验证要点

#### 3.1 常用贵金属检测方法

| 方法 | 适用标准 | 适用范围 | 验证要点 |
|---|---|---|---|
| **火试金法** | GB/T 9288 | 贵金属首饰含金量测定 | ①铅试金与标样比对；②灰吹温度控制；③分金操作规范性；④空白试验 |
| **X射线荧光光谱法（XRF）** | GB/T 21198.1~6 | 贵金属首饰元素分析 | ①校准曲线建立与验证；②不同基体效应校正；③表面镀层影响评估；④测试窗口选择 |
| **电感耦合等离子体质谱法（ICP-MS）** | GB/T 21198系列 | 痕量元素分析 | ①干扰校正；②内标选择；③检出限验证；④加标回收率 |
| **电感耦合等离子体发射光谱法（ICP-OES）** | 内部方法 | 贵金属元素定量 | ①波长选择；②基体匹配；③精密度和准确度验证 |
| **重量法** | GB/T 17832 | 银合金含银量测定 | ①沉淀条件优化；②沉淀完全性验证；③洗涤充分性 |

#### 3.2 贵金属检测特殊质控要求

| 质控项目 | 具体要求 |
|---|---|
| **标准物质** | 必须使用有证标准物质（CRM），如金/银/铂/钯国家标准物质 |
| **空白试验** | 每批样品进行空白试验，空白值应低于方法检出限 |
| **平行双样** | 每批样品至少10%进行平行双样测定，相对偏差应在允许范围内 |
| **加标回收** | 每批样品进行加标回收试验，回收率应在90%-110%范围内 |
| **样品代表性** | 对不均匀样品（如焊接首饰）应多点取样 |
| **表面镀层** | XRF法检测前应评估表面镀层对结果的影响 |

### 四、钻石分级的人员和环境要求

#### 4.1 人员资质要求

| 项目 | 具体要求 |
|---|---|
| **最低学历** | 珠宝鉴定、地质学、材料学等相关专业大专及以上学历 |
| **专业证书** | 须持有有效的钻石分级资格证书（DGA、HRD、FGA、GIA、NGTC等） |
| **实践经验** | 从事钻石分级工作满2年以上 |
| **继续教育** | 每年参加专业培训不少于16学时 |
| **能力维持** | 每年通过内部能力验证，结果满意 |
| **视力要求** | 无色盲、色弱，视力或矫正视力正常 |

#### 4.2 环境设施要求

| 环境参数 | 具体要求 | 监控方式 |
|---|---|---|
| **比色室** | 暗室环境，无杂散光 | 目视检查+光照度计 |
| **比色灯光源** | D65标准光源，色温5500-7200K | 色温计定期检测 |
| **比色背景** | 中性白色或灰色，无荧光 | 定期检查 |
| **比色槽** | 清洁，无灰尘和污渍 | 每次使用前清洁 |
| **标准比色石** | 齐全（D-N色级），有证书 | 定期核查 |
| **工作台** | 深色台面，无反光 | 目视检查 |
| **放大镜/显微镜** | 10倍放大镜，双目显微镜 | 定期校准 |

#### 4.3 钻石分级操作流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  1.样品接收  │ → │  2.重量称量  │ → │  3.颜色分级  │ → │  4.净度分级  │ → │  5.切工评价  │
│  (清洗、检查)│    │  (精确到0.0001g)│   │  (比色灯+比色石)│   │  (10×放大镜)  │    │  (比例仪+目视)│
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
                                                                                           ↓
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  9.报告签发  │ ← │  8.审核复核  │ ← │  7.荧光检测  │ ← │  6.比例测量  │ ← │  5.完成分级  │
│  (授权签字人) │    │  (技术负责人) │    │  (紫外灯下)   │    │  (切工仪)     │    │             │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

⚠️ **特别要求**：
1. 钻石分级应在独立的比色室内进行，避免外界光线干扰
2. 颜色分级时，比色石与待分级钻石的台面应朝上，亭部朝下
3. 净度分级应使用10倍放大镜，从多个方向观察
4. 切工评价应综合比例仪测量数据和目视评价
5. 分级结果应由2名以上有资质人员确认
6. 分级全过程应有详细记录，包括关键判断依据
