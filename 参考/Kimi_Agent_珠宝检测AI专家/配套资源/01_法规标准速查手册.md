## 一、2023版《检验检测机构资质认定评审准则》

### 1. 基本信息

| 项目 | 内容 |
|------|------|
| 发布时间 | 2023年5月30日（市场监管总局2023年第21号公告） |
| 实施日期 | **2023年12月1日** |
| 废止文件 | 《检验检测机构资质认定评审准则》（国认实〔2016〕33号） |
| 法律依据 | 《计量法》及其实施细则、《认证认可条例》 |
| 配套文件 | 《〈检验检测机构资质认定评审准则〉条文释义》（市监检测(司)函〔2023〕37号） |
| 一般程序审查表 | 《检验检测机构资质认定评审准则》一般程序审查（告知承诺核查）表 |

### 2. 核心结构（共四章21条）

```
第一章 总则（第1-6条）
├── 立法目的、适用范围、术语定义
├── 告知承诺核查规定
└── 评审工作原则

第二章 评审内容与要求（第7-11条）★核心章节
├── 第七条 — 机构主体（法律地位、独立性、公正性）
├── 第八条 — 人员（关键岗位任职资格、授权签字人）
├── 第九条 — 场所环境（固定场所、环境控制）
├── 第十条 — 设备设施（配备、校准、期间核查）
└── 第十一条 — 管理体系（建立、实施、保持）

第三章 评审方式与程序（第12-17条）
├── 现场评审、书面审查、远程评审
└── 评审程序：申请→受理→评审→整改→决定

第四章 附则（第18-21条）
└── 实施日期
```

### 3. 五大评审要素核心要求

| 要素 | 条款 | 核心要求摘要 |
|------|------|-------------|
| **机构主体** | 第七条 | 依法成立、独立承担法律责任；以公开方式自我承诺独立性与公正性；不得有不正当竞争行为；建立防范和惩治弄虚作假行为的机制 |
| **人员** | 第八条 | 技术负责人：中级及以上职称或同等能力；授权签字人：具有中级及以上职称或同等能力，经资质认定部门批准；质量负责人：有质量管理工作经验；人员培训、监督、能力确认 |
| **场所环境** | 第九条 | 具备固定工作场所；环境条件符合标准/技术规范要求；有效隔离不相容区域；具备环境监控记录 |
| **设备设施** | 第十条 | 配备满足要求的设备设施；量值溯源（校准/检定）；唯一性标识；使用记录；期间核查；标准物质管理 |
| **管理体系** | 第十一条 | 建立、实施、保持管理体系；覆盖所有活动；文件化；内部审核（每年至少1次）；管理评审（每年至少1次）；数据分析与持续改进；数据质量控制活动 |

### 4. 与旧版（2016版）的主要变化

| 变化维度 | 2016版 | 2023版 |
|---------|--------|--------|
| **法律依据** | 未明确列出具体法律法规 | 第一条明确列出《计量法》《认证认可条例》等法律依据 |
| **评审方式** | 以现场评审为主 | 建立"三位一体"评审体系：现场评审+书面审查+**远程评审** |
| **告知承诺** | 无专门条款 | 新增告知承诺现场核查规定（第五条） |
| **独立性** | 一般要求 | 明确要求以**公开方式进行自我承诺**，强调独立性、公正性、诚信度 |
| **人员要求** | 较笼统 | 细化授权签字人"同等能力"认定；明确关键岗位任职资格 |
| **数据管理** | 无特别要求 | 新增对**计算机信息系统管理**、**数据质量控制活动**的具体指引 |
| **风险管理** | 未强调 | 引入风险思维要求 |
| **管理体系** | 要求"建立" | 要求"有效、可控、稳定"实施 |

### 5. 珠宝检测领域的适用要点

⚠️ **重点提醒：**

1. **独立性承诺**：珠宝检测实验室涉及高价值样品，必须以公开方式做出独立性承诺，确保与珠宝生产、销售企业无利益关联
2. **人员资质**：珠宝鉴定人员需具备相应专业背景（地质、宝石学、材料科学等），授权签字人需中级及以上职称或同等能力
3. **设备溯源**：折射仪、分光镜、红外光谱仪、X射线荧光光谱仪等关键设备必须定期校准并溯源
4. **环境控制**：珠宝检测对光源、温湿度敏感，需确保鉴定环境符合标准要求
5. **样品管理**：贵重珠宝样品需建立严格的接收、保管、返还程序，确保样品安全

### 3. 各模块关键条款摘要

#### 模块一：4 通用要求

| 条款 | 标题 | 核心要求 |
|------|------|---------|
| 4.1 | 公正性 | 实验室应公正实施实验室活动，不受来自商业、财务或其他压力的不当影响；持续识别威胁公正性的风险 |
| 4.2 | 保密性 | 实验室通过法律上可执行的承诺，对在实验室活动中获取的所有信息负有保密责任 |

#### 模块二：5 结构要求

| 条款 | 核心要求 |
|------|---------|
| 5.1 | 实验室或其母体机构应是法定机构登记注册的法律实体 |
| 5.2 | 实验室管理层对实验室活动全面负责，指定质量负责人、技术负责人 |
| 5.3 | 清晰界定实验室活动范围，包括能力界限 |
| 5.4 | 建立与活动相适应的组织结构并文件化 |
| 5.5 | 明确各岗位职责、权限及相互关系 |
| 5.6 | 确保有足够的资源实施实验室活动 |

#### 模块三：6 资源要求

| 条款 | 标题 | 核心要求 |
|------|------|---------|
| 6.2 | 人员 | 人员应有所需的教育、资格、培训、技术知识、技能和经验；建立人员能力确认、授权、培训、监督机制 |
| 6.3 | 设施和环境条件 | 设施和环境条件应适合实验室活动，不应对结果有效性产生不利影响；监控、控制和记录环境条件 |
| 6.4 | 设备 | 设备应经校准、核查和维护；唯一性标识；使用记录；期间核查；不合格设备处理 |
| 6.5 | 计量溯源性 | 测量结果应溯源至国际单位制（SI）；校准证书应包含测量不确定度和/或符合性声明 |
| 6.6 | 外部提供的产品和服务 | 对影响实验室活动的外部产品和服务进行评价、选择、监控，并保存记录 |

#### 模块四：7 过程要求

| 条款 | 标题 | 核心要求 |
|------|------|---------|
| 7.1 | 要求、标书和合同的评审 | 建立合同评审程序，确保有能力满足客户要求 |
| 7.2 | 方法的选择、验证和确认 | 优先选用标准方法；非标方法需确认；方法验证/确认记录 |
| 7.4 | 检测或校准物品的处置 | 样品的全程管理：标识、存储、保留/处置，避免损坏或变质 |
| 7.5 | 技术记录 | 记录应包含足够信息以复现实验室活动；原始数据不可擦除；保存不少于6年 |
| 7.6 | 测量不确定度的评定 | 对校准实验室必须评定；对检测实验室，当客户要求或不确定度影响合格判定时应评定 |
| 7.7 | 结果有效性的保证 | 质量控制活动：定期能力验证、实验室内比对、质控样品、重复检测、审核报告等 |
| 7.8 | 报告结果 | 报告内容应准确、清晰、明确；符合特定方法或客户要求；报告签发 |
| 7.10 | 不符合工作 | 不符合工作的识别、控制、纠正及记录 |

#### 模块五：8 管理体系要求

| 条款 | 标题 | 核心要求 |
|------|------|---------|
| 8.1 | 方式A/方式B | A：按ISO/IEC 17025全部条款建立管理体系；B：按ISO 9001建立体系后，再满足17025的技术要求 |
| 8.2 | 管理体系文件 | 方针、目标、质量手册、程序文件、作业指导书、记录表格 |
| 8.5 | 应对风险和机遇的措施 | 识别可能影响结果有效性的风险和机遇，并采取措施 |
| 8.8 | 内部审核 | 每年至少1次，覆盖所有要素和活动 |
| 8.9 | 管理评审 | 每年至少1次，由管理层主持，输入全面、输出改进决策 |

### 4. 与CMA评审准则的对应关系

GB/T 27025-2019是实验室能力建设的通用技术标准，而2023版CMA评审准则是中国资质认定的行政许可技术评审依据。两者理念相通，但CMA评审准则更具中国监管特色。

| 10 | 风险管理导向 | 整体强化风险思维，要求识别影响结果有效性的风险 |

### 4. 关键条款摘要（珠宝检测重点关注）

| 条款 | 核心要求 | 珠宝检测实施要点 |
|------|---------|-----------------|
| 6.2 人员 | 检测人员需相关专业大专及以上学历；或10年以上相关经历 | 珠宝鉴定人员应具备宝石学、地质学、材料学等专业背景；定期进行盲样考核 |
| 6.3 设施环境 | 租借期≥2年；有完全支配权；相互干扰区域需隔离 | 显微镜观察区需避光、防震；光谱仪区需恒温恒湿；样品存储区需安保措施 |
| 6.4 设备 | 设备校准证书需含不确定度；定期期间核查 | 红外光谱仪、紫外-可见光谱仪、XRF等需每年校准；折射仪、天平需定期检定 |
| 7.2 方法 | 标准方法需验证；非标准方法需确认 | GB/T 16552/16553/16554为优先选用方法；红外/紫外光谱法需进行方法验证 |
| 7.7 结果有效性 | 定期参加能力验证（PT）；实验室内比对 | 参加珠宝玉石鉴定、贵金属含量测定等PT计划；定期进行人员间比对 |

## 四、CNAS-CL01-A015:2018珠宝检测领域应用说明

### 1. 基本信息

| 项目 | 内容 |
|------|------|
| 文件名称 | 《检测和校准实验室能力认可准则在珠宝玉石、贵金属检测领域的应用说明》 |
| 文件编号 | CNAS-CL01-A015:2018 |
| 发布日期 | 2018年3月1日 |
| 实施日期 | **2018年9月1日** |
| 替代文件 | CNAS-CL24:2015 |
| 附表版本 | CNAS-AL01-28（2020年8月31日更新） |
| 配套使用 | 与CNAS-CL01:2018同时使用；贵金属化学分析另需满足A002要求 |

### 2. 适用范围

> 适用于**珠宝玉石及贵金属检测领域**的实验室认可，具体包括：
> - 珠宝玉石的**鉴定**、**分级**、**性能测试**
> - 贵金属材料、成品的**含量/纯度**（无损检测手段）、**外观**及**性能**检测
> - 贵金属的**化学分析**需同时满足CNAS-CL01-A002《化学检测领域的应用说明》

⚠️ **注意**：贵金属无损检测指在不损害或不影响被检测对象使用性能的前提下，利用物理特性、光学特性等进行定性、定量分析的检测技术。

### 3. 核心要求摘要

#### （1）环境条件要求

### 3. GB/T 16554-2017《钻石分级》

| 项目 | 内容 |
|------|------|
| 标准号 | GB/T 16554-2017 |
| 替代标准 | GB/T 16554-2010 |
| 性质 | **国家标准（推荐性）** |
| 适用范围 | 天然的未镶嵌及镶嵌抛光钻石的分级 |

**核心内容（4C分级）：**

| **CNAS现场评审** | **2个月内**（一般45个自然日内） | CNAS-RL01 | 整改材料提交后，CNAS视情况决定是否需书面验证或再次现场复评 |
| **CNAS监督评审** | 2个月内 | CNAS-RL01 | 与初次/复评审要求一致 |
| **CMA/CNAS不符合项** | 按评审组要求 | 评审组通知 | 严重不符合项通常要求立即采取纠正措施 |

⚠️ **整改工作要点**：
1. **原因分析**：找到根本原因，而非表面原因
2. **纠正措施**：针对原因制定有效的纠正措施
3. **证据材料**：提供整改前后的对比证据（照片、记录、文件等）
4. **有效性验证**：验证纠正措施是否有效
5. **举一反三**：检查是否存在类似问题

## 七、⚠️ 常见违规风险与提醒

### 高风险事项清单

| 序号 | 风险事项 | 可能后果 | 预防措施 |
|------|---------|---------|---------|
| 1 | **超范围出具报告**（CMA/CNAS未授权的项目） | 行政处罚、证书暂停 | 定期核查资质范围，新增项目及时扩项 |
| 2 | **授权签字人超范围签发** | 报告无效、证书暂停 | 明确每位授权签字人的授权领域 |
| 3 | **使用过期标准方法** | 报告被质疑、客户投诉 | 建立标准查新机制，每季度查新一次 |
| 4 | **设备超期未校准** | 数据无效、结果不被采信 | 建立设备校准计划，提前1个月提醒 |
| 5 | **记录保存不完整/不足6年** | 无法追溯、评审不通过 | 建立记录管理台账，定期归档检查 |
| 6 | **未参加能力验证** | CNAS警告、证书暂停 | 每年初制定PT计划，按时参加 |
| 7 | **内审/管评未按时进行** | 不符合项、证书暂停 | 年初制定审核计划，严格执行 |
| 8 | **珠宝定名不规范**（未按GB/T 16552） | 客户投诉、法律风险 | 定期培训标准更新内容，报告审核严格把关 |
| 9 | **贵金属命名违规**（使用"千足金"等废止名称） | 行政处罚 | 严格执行GB 11887-2012第1号修改单 |
| 10 | **样品管理疏漏**（贵重样品丢失/损坏） | 重大经济损失、声誉损害 | 建立样品全程跟踪系统，贵重样品专人专管 |
