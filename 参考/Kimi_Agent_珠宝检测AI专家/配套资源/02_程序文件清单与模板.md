#### A1. 保护客户机密信息和所有权程序

| 项目 | 内容 |
|:---|:---|
| **文件编号** | PROC-001 |
| **对应条款** | RB/T 214 第4.1.4条 / CL01 第4.2条 |
| **核心目的** | 保护客户在珠宝检测过程中的机密信息、所有权及知识产权 |
| **珠宝检测特殊要求** | 💎 珠宝样品往往具有高价值和隐私性，需建立分级保密机制 |

**要点内容框架**：
1. 保密信息分级（一般/重要/绝密）
2. 保密协议签署范围（全体员工、临时人员、分包方）
3. 样品信息保密（送检人信息、鉴定结果提前泄露控制）
4. 检测数据保密（电子数据访问权限控制）
5. 客户身份保密（匿名送检处理流程）
6. 违规处罚措施

**关键提醒**：⚠️ 珠宝送检客户对保密性要求极高，尤其是涉及名人、拍卖行、保险公司等客户。

#### 特殊要求增补

| 管理环节 | 标准要求 | 珠宝特殊强化措施 |
|:---|:---|:---|
| **接收** | 正常登记 | 由两名人员共同接收、核对、拍照 |
| **标识** | 唯一性编号 | 加贴"高价值"红色警示标签 |
| **保管** | 样品柜 | 💎 存入保险柜（双人双锁），24小时监控 |
| **流转** | 交接签字 | 💎 全程双人陪同，填写《高价值样品流转单》 |
| **检测** | 正常检测 | 优先安排检测，减少在实验室滞留时间 |
| **归还** | 签收确认 | 💎 由两名人员共同核对后归还，全程录像 |

#### 工作程序要点

```
4.A.1 高价值样品识别
    由接样人员在委托单上标注"高价值"，通知样品主管。

4.A.2 专项登记
    填写《高价值样品登记表》，记录：样品描述、估计价值、
    客户特殊要求、照片编号。

4.A.3 保险柜保管
    非检测时间必须存放于保险柜中。保险柜密码由样品主管和
    质量负责人分别掌握。

4.A.4 检测过程控制
    检测时由检测人员和监督人员同时在场。检测完成后立即
    归还保险柜。

4.A.5 流转追踪
    每次流转填写《高价值样品流转单》，记录：取出时间、
    取出人、目的、归还时间、归还人、确认人。

4.A.6 异常处理
    如发生样品遗失、损坏，立即启动应急预案：
    1) 立即报告质量负责人和最高管理者
    2) 保护现场，查看监控记录
    3) 2小时内书面报告客户
    4) 配合保险公司调查
    5) 保留所有证据材料

4.A.7 保险要求
    实验室应购买样品综合保险，确保高价值样品在保管期间
    的风险覆盖。
```

### 3.2 钻石分级质量控制程序

**建议处理方式**：作为"结果有效性保证程序（PROC-022）"的子程序，编号 PROC-022-A

#### 适用范围
适用于钻石的4C分级（克拉重量Carat、颜色Color、净度Clarity、切工Cut）质量控制。

#### 特殊要求增补

| 分级项目 | 质控方式 | 频次 | 判定标准 |
|:---|:---|:---:|:---|
| **颜色分级** | 与标准比色石比对 | 每日开工前 | 偏差不超过1个色级 |
| **颜色分级** | 人员比对（盲评） | 每月至少1次 | 一致性≥90% |
| **净度分级** | 标准样品测试 | 每周至少1次 | 偏差不超过1个净度级 |
| **净度分级** | 人员比对（盲评） | 每月至少1次 | 一致性≥85% |
| **切工评价** | 设备比对（不同仪器） | 每季度1次 | 偏差在允许范围内 |
| **克拉重量** | 标准砝码核查 | 每日开工前 | 偏差≤0.001ct |

#### 工作程序要点

```
4.A.1 比色石管理
    4.A.1.1 比色石应定期溯源至国际/国家标准（GIA/HRD比色石）
    4.A.1.2 比色石使用前后应清洁，存放于专用盒中
    4.A.1.3 比色石每年至少一次由外部机构核查

4.A.2 颜色分级日常质控
    4.A.2.1 每日开工前，分级师用标准比色石进行自检
    4.A.2.2 记录比色石比对结果
    4.A.2.3 如比对结果超差，应停止分级，查明原因

4.A.3 人员比对程序
    4.A.3.1 由质量负责人组织，选取5-10颗不同色级的样品
    4.A.3.2 参与比对的每位分级师独立进行盲评
    4.A.3.3 统计比对结果，计算一致性
    4.A.3.4 一致性不达标时，进行培训和再比对

4.A.4 净度分级质控
    4.A.4.1 使用标准净度样品进行日常核查
    4.A.4.2 定期与参考实验室进行比对

4.A.5 重量测量质控
    4.A.5.1 每日开工前用标准砝码校准天平
    4.A.5.2 记录校准结果
    4.A.5.3 千分之一天平每月期间核查
```

### 3.3 贵金属检测期间核查程序

**建议处理方式**：作为"设备管理程序（PROC-006）"或"计量溯源性控制程序（PROC-007）"的子程序，编号 PROC-007-A

#### 适用范围
适用于X射线荧光光谱仪（XRF）、电感耦合等离子体发射光谱仪（ICP-OES）等贵金属检测设备的期间核查。

#### 期间核查要求

| 设备类型 | 期间核查方法 | 频次 | 判定标准 |
|:---|:---|:---:|:---|
| XRF光谱仪 | 使用标准金片/银片测量 | 每周至少1次 | 测量值与标准值偏差≤允许误差 |
| 电子天平 | 使用标准砝码核查 | 每日开工前 | 示值误差在最大允许误差内 |
| ICP-OES | 使用标准溶液核查 | 每次开机/每批次 | 标准溶液回收率在95%-105% |
| 火试金设备 | 使用标准样品核查 | 每月1次 | 结果在允许误差范围内 |

#### 工作程序要点

```
4.A.1 XRF期间核查程序
    4.A.1.1 准备标准金片（Au含量已知）
    4.A.1.2 开机预热30分钟后进行核查
    4.A.1.3 将标准金片置于测量位置
    4.A.1.4 按操作规程测量3次，取平均值
    4.A.1.5 计算测量值与标准值的偏差
    4.A.1.6 偏差在允许范围内则设备可用
    4.A.1.7 偏差超差则停止使用，查明原因

4.A.2 判定准则
    |Au含量(‰)|允许偏差(‰)|
    |≥999|±0.5|
    |≥990|±1.0|
    |≥750|±2.0|
    |≥585|±3.0|

4.A.3 异常处理
    期间核查超差时：
    1) 立即停止该设备检测工作
    2) 检查设备状态（X射线管、探测器、光路）
    3) 检查校准曲线是否需要重新建立
    4) 联系维修或校准
    5) 重新校准/维修后重新核查合格方可使用
    6) 追溯上次核查合格以来的检测结果
```

### 3.4 主观判断项目人员比对程序

**建议处理方式**：作为"人员监督程序（PROC-004）"或"结果有效性保证程序（PROC-022）"的子程序，编号 PROC-022-B

#### 适用范围
适用于以主观判断为主要依据的检测项目，包括：
- 钻石颜色分级
- 钻石净度分级
- 翡翠颜色评价
- 珍珠光泽度评价
- 其他依赖人眼判断的项目

#### 人员比对方案

| 比对项目 | 样品数量 | 参与人数 | 评价方式 | 合格标准 |
|:---|:---:|:---:|:---|:---|
| 钻石颜色分级 | 5-10颗（覆盖D-J色） | ≥3人 | 盲评，计算Kappa系数 | Kappa≥0.8 |
| 钻石净度分级 | 5-10颗（FL-SI范围） | ≥3人 | 盲评，计算Kappa系数 | Kappa≥0.7 |
| 翡翠颜色评价 | 5-8件 | ≥3人 | 盲评 | 一致性≥80% |
| 珍珠光泽度 | 5-8件 | ≥3人 | 盲评 | 一致性≥80% |

#### 工作程序要点

```
4.A.1 比对计划
    每年初制定年度人员比对计划，明确：
    - 比对项目
    - 计划时间
    - 参与人员
    - 评价方法

4.A.2 比对准备
    4.A.2.1 选择比对样品（覆盖不同等级）
    4.A.2.2 样品由组织者编号，参与者不知晓真实等级
    4.A.2.3 准备比对环境（标准光源、暗箱等）
    4.A.2.4 准备记录表格

4.A.3 比对实施
    4.A.3.1 参与者独立进行判断，不得互相讨论
    4.A.3.2 组织者记录每位参与者的判断结果
    4.A.3.3 比对应在相同环境条件下进行

4.A.4 结果统计
    4.A.4.1 汇总各参与者的判断结果
    4.A.4.2 与参考值比对，计算偏差
    4.A.4.3 计算人员间的一致性（Kappa系数）
    4.A.4.4 生成比对报告

4.A.5 结果处理
    4.A.5.1 一致性达标：记录归档
    4.A.5.2 一致性不达标：
        a) 分析原因（人员技能/标准理解/视力因素）
        b) 安排针对性培训
        c) 1个月内重新比对
        d) 仍不达标则暂停该项目授权

4.A.6 Kappa系数计算方法
    Kappa = (Po - Pe) / (1 - Pe)
    Po = 观察到的一致性比例
    Pe = 期望的一致性比例（随机一致）
    Kappa值解释：
    - ≥0.8：几乎完全一致
    - 0.6-0.8：高度一致
    - 0.4-0.6：中度一致
    - <0.4：一致性差
```
