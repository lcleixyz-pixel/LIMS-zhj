## 生成规则说明

- `导入`：目录基准内存在对应空白模板，或封面/目录类元文件，可作为受控记录表格模板进入文件控制。
- `人工确认`：目录外文件、附件缺失、重复编号、设备/样品专用模板、文件名与目录名称不一致或归属边界不清的附件。
- `跳过`：含年份、已填、具体样品编号、培训记录子目录、使用中清单、整改资料等历史记录特征的文件。
- `匹配结论` 结合 Kimi 记录表格清单、TB 内审/管评模板和名称规则生成，最终导入前仍建议由质量负责人复核。
- 正式导入建议状态仍按既定策略设为 `published`；本轮仅生成预览，不复制附件、不写数据库。
